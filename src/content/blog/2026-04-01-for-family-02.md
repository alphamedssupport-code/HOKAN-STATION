## 「退院して家に帰る」が決まったら

入院中の親が「退院して自宅に帰ることになった」「在宅療養を希望している」というとき、ご家族はさまざまな不安を感じるのではないでしょうか。

- 「自宅で本当にケアできるのか？」
- 「夜中に急に悪くなったらどうしよう」
- 「何から準備すればいいかわからない」

こうした不安は多くの介護家族が経験するものです。しかし、**適切な準備と在宅サービスの活用**によって、在宅療養は家族にとっても患者にとっても安心のできる選択肢になります。

## ステップ1：退院前カンファレンスに積極的に参加する

在宅療養の準備は**病院在籍中から始まります**。多くの病院では「退院前カンファレンス」が開催されます。

### カンファレンスで確認すべきこと

**医療・看護面**
- どんな医療処置が自宅でも必要か（点滴・吸引・胃瘻管理など）
- 服薬内容と飲み方（何を・いつ・どのくらい）
- 主治医の名前・連絡先・訪問診療の有無
- 緊急時に連絡すべき窓口はどこか

**介護・生活面**
- 日常生活でどのくらい介助が必要か（食事・排泄・入浴）
- 自宅での移動・体位変換の注意点
- 食事形態（普通食・刻み食・とろみ食）

**在宅サービスの調整**
- 訪問看護・訪問介護の開始スケジュール
- 担当ケアマネジャーの決定
- 必要な医療機器・福祉用具のレンタル手続き

## ステップ2：自宅環境を整える

### バリアフリー化・安全対策

**手すりの設置**
- 玄関の上がり框（かまち）
- 廊下・トイレ・洗面所・浴室
- ベッドからの立ち上がり場所

**段差の解消**
- 玄関・廊下・トイレ入口の段差にスロープを設置

**転倒リスクの排除**
- 部屋中の不要な家具・コード類を整理
- 滑りやすい場所に滑り止めマットを設置
- 夜間のトイレ動線に足元ライトを設置

### 医療機器・衛生用品の準備

| 用品 | 用途 |
|---|---|
| 血圧計・体温計・パルスオキシメーター | バイタルサイン管理 |
| 吸引器 | 痰の吸引が必要な場合 |
| 酸素濃縮器 | 在宅酸素療法が必要な場合 |
| 尿器・ポータブルトイレ | 排泄の自立支援 |
| 防水シーツ・使い捨て手袋 | 排泄ケア・処置時 |
| 体位変換クッション | 褥瘡予防 |

医療機器の多くは介護保険・医療保険でレンタルできます。

## ステップ3：在宅サービスの手続きを行う

### 介護保険の申請

在宅介護サービスを利用するためには**介護保険の要介護認定**が必要です。

**申請の流れ**
1. 市区町村の介護保険担当窓口または地域包括支援センターへ申請
2. 調査員が自宅を訪問（認定調査）
3. 主治医意見書の作成
4. 認定結果の通知（申請から約1か月）

申請から認定まで時間がかかるため、**退院前から早めに申請**することをおすすめします。

### ケアマネジャーの決定・ケアプランの作成

**ケアマネジャーの探し方**
- 病院の地域連携室（退院支援看護師・MSW）に紹介してもらう
- 市区町村の介護保険担当窓口で居宅介護支援事業所の一覧を入手

### 訪問看護の利用開始

**訪問看護を開始するための流れ**
1. 主治医が「訪問看護指示書」を作成
2. 訪問看護ステーションとの契約
3. 担当看護師によるアセスメント訪問
4. 訪問スケジュールの決定・サービス開始

## ステップ4：緊急時の対応を確認する

### 緊急連絡先リストの作成

以下の連絡先を見やすい場所（冷蔵庫・電話のそば）に張り出しておきます。

| 相手 | 連絡すべきタイミング |
|---|---|
| 訪問看護ステーション（24時間対応） | 症状変化・処置トラブル |
| 主治医・訪問診療クリニック | 病状変化の相談 |
| 担当ケアマネジャー | サービスの変更・相談 |
| 救急（119） | 意識障害・呼吸困難など |

### 「こんなときどうする？」を事前に確認

- **発熱が出たら**：体温を測り、訪問看護師に連絡
- **転倒・骨折が疑われるとき**：無理に動かさず、救急または訪問看護師へ連絡
- **息苦しそうにしているとき**：SpO2測定、訪問看護師・救急への連絡を判断

### 看取りの希望を確認しておく

在宅での看取りを希望する場合、事前に家族・本人・主治医で**「人生の最終段階の医療・ケアについての希望」**を話し合っておくことが重要です（ACP：アドバンス・ケア・プランニング）。

## ステップ5：家族間で介護を分担する

### 介護役割の分担を話し合う

- 主介護者（最も多くの時間を担う人）
- 日中・夜間の担当分担
- 経済的な役割（費用の負担）
- 緊急時の連絡・対応の担当

### 離れて住む家族の関わり方

- 定期的な電話・ビデオ通話で状況を確認
- 金銭的なサポート
- 週末などに帰省して、主介護者を休ませる
- サービスに関する調査・手続きの代行

## まとめ

在宅療養の開始は、準備なしに始めると家族全員が疲弊してしまうことがあります。しかし、事前に環境・サービス・緊急時の対応を整えておくことで、在宅での療養生活は安心・安全なものになります。

**一人で抱え込まないこと**が最も大切なメッセージです。病院の退院支援スタッフ、ケアマネジャー、訪問看護師など、多くの専門家が家族をサポートします。遠慮なく相談し、チームで在宅療養を支えていきましょう。
