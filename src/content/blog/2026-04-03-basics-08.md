## イントロダクション

訪問看護を受ける患者さんやご家族が抱える疑問は多いものです。「訪問看護師は何を記録しているのか」「看護計画はどのように立てられるのか」「どうやって医師に報告しているのか」──こうした質問は当然の疑問です。実は、訪問看護の記録と報告は、患者さんの安全で質の高いケア提供の土台となる、極めて重要なプロセスです。本記事では、看護計画の策定から日々の訪問記録、主治医との連携報告まで、訪問看護の記録・報告システムの仕組みを現場視点で詳しく解説します。正確な記録がどのようにあなたのケアを守り、改善につながるのかを理解することで、より信頼できる訪問看護サービスを受けるための知識が身につきます。

## 訪問看護における記録と報告の役割

### なぜ記録と報告が重要なのか

訪問看護における記録と報告は、単なる「義務」ではなく、患者さんの療養を支える最も重要な基盤です。

**記録・報告の3つの主要な役割：**

1. **患者安全の確保**
   - 健康状態の変化を経時的に把握
   - 医療事故・インシデントの防止
   - 複数の医療職間での情報共有

2. **法的責任と証拠保持**
   - 診療報酬請求の根拠
   - 医療関連訴訟における重要な証拠
   - 適切なケア提供の記録

3. **ケア質の向上**
   - 患者さんの変化傾向を可視化
   - 看護介入の効果測定
   - 看護計画の改善・修正

厚生労働省の「訪問看護・訪問リハビリテーション実施指導」では、訪問看護ステーションに対して「患者ごとに記録を整備し、5年間保存すること」が定められています。これは、あなたのケア内容が適切に記録され、管理されていることを意味しています。

### 訪問看護の記録に求められる基準

看護記録が備えるべき基準として、日本看護協会は「CARER」という枠組みを提唱しています：

| 項目 | 説明 | 具体例 |
|------|------|--------|
| **C（Complete）** | 完全性 | 必要な情報が漏れなく記載 |
| **A（Accurate）** | 正確性 | 測定値や観察内容が事実のまま |
| **R（Relevant）** | 関連性 | 患者さんの目標と関連する内容 |
| **E（Explicit）** | 明確性 | 曖昧さなく、第三者が理解可能 |
| **R（Readable）** | 可読性 | 判読困難な記載がない |

訪問看護師は、これらの基準に基づいて、日々の記録をつけています。

## 看護計画の策定プロセス

### 初回面談と情報収集

訪問看護が開始されるとき、訪問看護師はまず**アセスメント（評価）**を行います。これが看護計画の第一歩です。

**初回訪問で実施される主な項目：**

1. **基本情報の収集**
   - 患者さんの年齢、性別、家族構成
   - 主な医学的診断名（疾患）
   - 現在使用している薬剤
   - アレルギー情報
   - 主治医の医学的指示内容

2. **身体的アセスメント**
   - バイタルサイン（体温、血圧、脈拍、呼吸数、酸素飽和度）
   - 身体診察（呼吸音、皮膚の状態、浮腫など）
   - ADL（日常生活動作）の評価：食事、排泄、入浴、着替えなど
   - 認知機能の評価（意識レベル、見当識など）

3. **心理社会的アセスメント**
   - 患者さん・ご家族の不安や悩み
   - 社会的背景（仕事、経済状況、生活環境）
   - 精神面の状態（うつ傾向の有無など）
   - 生活上の優先順位

4. **生活環境の確認**
   - 居住環境（バリアフリー状況、段差の有無）
   - 家族の介護能力
   - 近隣の支援体制
   - 公共交通のアクセス

### NANDA-I看護診断による問題の整理

収集した情報から、訪問看護師は「NANDA-I看護診断」という国際的な枠組みを使用して、患者さんの看護上の問題を整理します。

**一般的な看護診断の例：**

- **活動不耐性**：運動・活動によって疲労・息切れが起こりやすい状態
- **皮膚統合性の障害リスク**：褥瘡（床ずれ）が生じるリスク
- **感染リスク**：尿路感染症や肺炎などが起こりやすい状態
- **知識不足**：患者さん・ご家族が疾患や自己管理について理解していない状態
- **社会的孤立のリスク**：外出機会の減少などで精神的孤立が起こりやすい状態

これらの診断は、単なる医学的診断（例：糖尿病）ではなく、その人の「生活のなかで今、何が課題なのか」を表現しています。

### 看護計画の構成要素

看護計画は、以下の4つの要素で構成されます：

**1. 目標（Goal）**
- 短期目標（2週間～1ヶ月）：具体的で測定可能
- 長期目標（3ヶ月以上）：患者さんが達成したい生活の状態

**例：** 
- 短期目標「3週間以内に、血糖値を朝食前100～130mg/dLの範囲に保つ」
- 長期目標「6ヶ月後に、自分で血糖測定と食事管理ができる」

**2. 看護介入（Nursing Intervention）**
- 患者さんを目標に導くための具体的な行動や指導
- 頻度・方法が明記される

**例：**
- 毎週月・水・金に血糖測定の実施状況と食事内容を確認する
- 月1回、栄養士による食事指導を実施する
- 自己測定結果を記録する手帳の使用方法を指導する

**3. 根拠（Rationale）**
- なぜその介入を行うのかについての医学的・看護学的背景

**例：**
- 週3回の確認は、短期的な血糖コントロール状況を把握し、早期に問題を発見するため
- 手帳使用は、患者さん自身が経時的な変化を認識し、自己管理意欲を高めるため

**4. 評価方法（Evaluation）**
- 計画がどの程度達成されたかを判断する方法

**例：**
- 血糖測定結果が目標範囲内で推移しているか
- 患者さんが自分で記録できているか
- 患者さんが食事療法を実行しているか

### 看護計画書の記載例

実際の訪問看護では、以下のようなフォーマットで看護計画が記録されます：

```
【患者名】山田太郎（75歳、男性）
【主診断】2型糖尿病、高血圧症
【看護診断】血糖自己管理知識不足

【短期目標】3週間以内に、患者が毎日の血糖測定を独力で実施できるようになる

【看護介入】
1. 初回訪問時に血糖測定機器の操作方法を説明・指導（所要時間30分）
2. 毎週月・水・金に訪問時、測定結果と測定方法を確認
3. 測定結果を記録する手帳の使用方法を指導
4. 困ったことがあれば24時間対応の相談窓口を周知

【根拠】
- 週3回の訪問間隔は、技術習得と不安解消に最適（看護学文献より）
- 手帳の使用は、患者の自己意識を高め、継続率を向上させる

【評価予定日】4月20日（初回から3週間後）
```

## 日々の訪問記録の書き方と記載項目

### 訪問記録に含むべき基本情報

訪問看護師が毎回の訪問後に作成する「訪問記録」には、法的要件と臨床的要件があります。

**訪問記録に必須の項目：**

| 項目 | 内容 | 記載のポイント |
|------|------|----------------|
| **訪問日時** | 実訪問日と時刻 | 何月何日の何時から何時まで |
| **訪問目的** | その訪問での主な目的 | 「血圧管理」「創処置」など |
| **バイタルサイン** | 体温、血圧、脈拍、呼吸数、酸素飽和度 | 測定時刻を明記 |
| **観察所見** | 患者さんの身体的・精神的状態 | 具体的で客観的な表現 |
| **実施内容** | 行った看護処置や指導 | 詳細で再現可能な記載 |
| **患者・家族の反応** | 患者さんの反応や理解度 | 言動・表情も含める |
| **問題・異常所見** | 懸念事項や予期しない変化 | 医師への報告内容との整合性 |
| **次回訪問予定** | 次の訪問日時と目的 | 予定変更の可能性も記載 |
| **指示者（医師）への報告内容** | 医師に伝えた・伝える内容 | 医師の指示有無も記載 |
| **署名** | 訪問看護師の氏名・押印 | 記録者の責任を明確化 |

### 適切な観察記録の具体例

**良い例と悪い例の比較：**

| 良い記録 | 問題のある記録 |
|---------|----------------|
| 「訪問時、患者は臥床状態。呼吸数24回/分と上昇。咳嗽あり。痰は黄色い粘性。酸素飽和度 92%（室内気）。以前の訪問（3日前）では呼吸数18回/分、酸素飽和度96%だった。患者は『息がしずらい感じがする』と訴えている。医師に報告して指示を仰いだ結果、肺炎の可能性として血液検査と胸部X線を指示された。本日は指示に従い採血補助を実施した。」 | 「呼吸がおかしい」「状態が悪い」「医師に連絡した」 |
| **利点：** 時系列・具体的な数値・変化傾向・医師の指示・実施内容が明確 | **問題：** 曖昧で、他職種が理解困難。法的証拠として不十分 |

**記録作成時の黄金ルール：**

1. **5W1Hで記述する**
   - **Who（誰が）** ：患者さんの行動や発言
   - **What（何を）** ：実施した看護内容
   - **When（いつ）** ：日時を正確に
   - **Where（どこで）** ：訪問場所や身体部位
   - **Why（なぜ）** ：その判断の根拠
   - **How（どのように）** ：具体的な方法や患者反応

2. **客観的事実を記載する**
   - 推測や解釈ではなく、実際に見た・聞いた・測定した内容
   - 「患者は疲れているようだ」 ❌ 
   - 「患者は会話中、数回ため息をつき、『疲れてしまった』と述べた」 ⭕

3. **正確な医学用語を使う**
   - 曖昧な言葉を避ける
   - 「咳をしている」より「乾性咳嗽が1日3～4回」と具体的に

4. **訂正は二重線で**
   - 間違った場合、修正液は使わずボールペンで二重線を引き、その上に正しい記載
   - 日付と訂正者の署名も必須
   - 電子記録の場合
