## イントロダクション

病院での勤務経験を活かしながら、より患者さんと向き合える仕事をしたい。夜勤の負担を減らしたい。地域に密着した医療を学びたい。そのような理由で、訪問看護師への転職を検討している看護師は少なくありません。しかし実際の転職では、「病院と訪問看護の仕事内容の違いは何か」「どうやって転職活動を進めるのか」「入職後、現場でやっていけるのか」といった不安や疑問が生じるものです。

本記事では、実際に病院から訪問看護への転職を経験した看護師の視点から、転職の流れ・必要な準備・現場での適応ポイントまで、具体的かつ実践的な情報をお伝えします。あなたのキャリアの選択肢を広げるための参考情報として、ぜひご活用ください。

## 1. 病院看護師と訪問看護師の仕事内容の違いを理解する

### 1-1. 職場環境と業務スタイルの比較

**病院看護師の特徴：**
- 複数の患者さんを同時に担当（一般病棟で5～10人）
- チーム医療が中心（医師、薬剤師、栄養士など多職種と連携）
- 決められた勤務時間（交替勤務や夜勤あり）
- 医療機器が充実した環境
- 急性期から慢性期まで多様な病態に対応
- 厳密な指示命令系統と電子カルテシステム

**訪問看護師の特徴：**
- 1回の訪問あたり1世帯（患者さんと家族が対象）
- 患者さんの生活の場（自宅やグループホーム）で行われる
- 裁量を持った自律的な判断が求められる
- 医療機器は限定的だが、創意工夫が活かされる
- 在宅での日常生活支援が中心
- 多職種連携（ケアマネジャー、薬剤師、医師など）は遠隔・書面が多い

### 1-2. やりがいと課題の違い

**病院看護師のやりがい：**
- 急性期の患者さんの回復を直接サポートできる
- チームの一員として組織的に動ける
- 新しい医療技術や治療法を学ぶ機会が多い
- 先輩のサポート体制が整っている傾向

**病院看護師の課題：**
- 患者さんとの関係が短期間（入院期間のみ）
- 夜勤による身体的・心理的負担
- 多数の患者さんを同時に担当するため個別対応の時間が限定的
- 急性期の対応に追われる

**訪問看護師のやりがい：**
- 患者さんの回復から最期まで、長期的に寄り添える
- 患者さんと家族の信頼関係が深まりやすい
- 生活を支える看護ができる充実感
- 自らの判断と創意工夫が直接成果につながる
- 身体的な負担（夜勤がない）が少ない事業所が多い

**訪問看護師の課題：**
- 1人で判断する場面が多く、責任が大きい
- 緊急対応が予測困難（夜間対応がある場合がある）
- 多くの事業所で新人教育体制が十分でない
- 運転（移動時間）の負担
- 電子カルテの操作性がまちまち

## 2. 転職前に確認すべきポイント

### 2-1. 自分のキャリアプランと適性の確認

訪問看護師への転職を決める前に、以下の項目について自問自答することをお勧めします。

**チェックリスト：あなたは訪問看護に向いているか**

- [ ] 患者さんと家族の生活背景に興味を持てるか
- [ ] 1人で判断・対応する裁量を望むか
- [ ] 自動車運転ができるか（多くの事業所で必須）
- [ ] 自宅訪問時の感染対策や衛生管理を厳密に実行できるか
- [ ] 急変対応が得意で、冷静に判断できるか
- [ ] 夜勤や交替勤務から離れたいか
- [ ] 患者さんと長期的な関係を築きたいか

上記項目のうち、特に「自動車運転」「1人での判断」「長期関係の構築」の3つは重要です。これらが大きなストレスになると感じる場合は、転職前に十分な検討が必要です。

### 2-2. 看護技術のギャップを知る

病院では習得できたが、訪問看護で必要とされる技術には以下のようなものがあります。

| 技術 | 病院での状況 | 訪問看護での特徴 |
|------|-----------|-----------------|
| **創傷処置** | ガーゼ交換が中心、無菌操作が厳密 | 清潔操作が中心、家庭環境での工夫が必要 |
| **排泄ケア** | トイレ環境が整備されている | 自宅の便座・ポータブルトイレの工夫 |
| **栄養管理** | 栄養科と連携、食事形態が統一 | 患者さん・家族の希望と栄養バランスの調整 |
| **褥瘡対策** | 医師の指示が即座に得られる | 予防と初期対応が中心、医師判断の判定難度が高い |
| **リハビリテーション** | リハビリ科と連携 | セルフケアと環境調整が中心 |
| **精神的ケア** | 患者さん・家族との接触時間が限定的 | 生活の大部分を支援するため、心理的課題への対応が重要 |

特に「創傷処置（傷の手当て）」「褥瘡（床ずれ）対策」「栄養管理」については、病院での経験をそのまま活かすのではなく、**在宅環境に合わせたアプローチが必要**です。

### 2-3. 給与・待遇の相場確認

訪問看護ステーション（在宅医療事業所）の給与は、病院と比較して以下の特徴があります。

**2024年度の相場データ（厚生労働省「医療経済実調査」参考）**
- 訪問看護師の平均月給：30～35万円（病院看護師の35～40万円と比較)
- 夜勤手当がない分、月給が低くなる傾向
- 訪問回数や訪問距離による手当があるかないか、事業所差が大きい
- ボーナスの有無・支給額が事業所によって大きく異なる

**確認項目：**
- 基本給
- 訪問手当（訪問1件あたり）
- 交通費（全額支給か一部か、あるいは自己負担か）
- 夜間・深夜対応手当（緊急対応がある場合）
- ボーナス支給の有無と支給月数
- 退職金制度の有無
- 健康保険・厚生年金加入の有無

給与は総合的に判断し、生活費の見直しを検討しましょう。

## 3. 転職活動の実践的なステップ

### 3-1. 情報収集と事業所選びのポイント

**訪問看護ステーション選びのチェックリスト**

1. **組織体制**
   - [ ] 常勤スタッフ数が十分か（目安：常勤5～10名以上）
   - [ ] 管理者が訪問看護の実務経験者か
   - [ ] 新人教育体制が整備されているか
   - [ ] 先輩スタッフに相談できる体制か

2. **教育・研修**
   - [ ] 入職時の研修プログラムがあるか
   - [ ] OJT（On-the-Job Training）の期間は何ヶ月か
   - [ ] 外部研修の補助制度があるか
   - [ ] 訪問同行期間は2週間以上か

3. **勤務条件**
   - [ ] 基本勤務時間は何時～何時か
   - [ ] 夜間対応（オンコール）の有無と頻度
   - [ ] 月の訪問件数目安（目安：1日4～6件）
   - [ ] 休日取得は可能か（週休2日制か）

4. **患者層と診療報酬基準**
   - [ ] 訪問診療の患者が多いか、医師の指示書ありの患者か
   - [ ] 介護保険利用者の割合
   - [ ] 末期がん患者など特定分野の専門性があるか
   - [ ] 利用者数は安定しているか

5. **働く環境**
   - [ ] 訪問に使用する車（軽自動車か普通自動車か、社用車か）
   - [ ] 電子カルテシステムは導入されているか
   - [ ] スタッフの離職率
   - [ ] 職場の人間関係（面接時の雰囲気から判断）

**情報収集の方法：**
- 求人サイト（マイナビ看護師、看護roo!、ナースではたらこ など）
- 訪問看護関連の業界情報サイト
- ハローワーク
- 知人の紹介（できれば訪問看護経験者）
- **面接時に現場スタッフに直接質問する**

### 3-2. 応募書類と面接での注意点

**職務経歴書で強調すべきポイント：**

病院での経験を、訪問看護で活かせる形で記述することが重要です。

❌ 避けるべき書き方：
「●●病院で5年間、内科病棟に配属され、医師の指示に基づいて看護を行いました」

✓ 推奨する書き方：
「●●病院内科病棟での5年間で、高血圧・糖尿病・心不全など慢性疾患患者の日常生活指導を経験。患者教育と自己管理支援に力を入れ、退院後の生活をサポートした経験を持ちます。この経験を、訪問看護での在宅療養者の生活支援に活かしたいと考えています」

**面接での想定質問と答え方のポイント：**

| 質問 | 回答のポイント |
|------|--------------|
| 「なぜ訪問看護を志望したのか」 | 病院での経験の限界と、訪問看護での成長機会を前向きに述べる。「患者さんとの関係を長期的に築きたい」など具体的な理由 |
| 「急変対応の経験は」 | 具体的な事例（患者ID等は除く）を1～2つ。判断フロー・対応内容を簡潔に説明 |
| 「1人で判断することへの不安は」 | 不安を認めつつ、「先輩への相談体制を活用したい」など前向きな姿勢を示す |
| 「自動車運転について」 | 運転免許保有を確認。長時間運転の体験があれば述べる |
| 「夜勤の経験」 | 夜勤がない場合は、「体調管理に工夫があった」など、自己管理能力をアピール |

**面接でこちらが質問すべき事項：**
- 「新人教育は具体的にどのように進めるのか、研修期間は」
- 「緊急時の判断ルール、医師への相談フローは」
- 「現在のスタッフは平均何年勤続しているか」
- 「月平均の訪問件数と1回の訪問時間は」
- 「訪問時の課題や困りごとについて、どうサポートしているか」

## 4. 入職前から入職後3ヶ月までの準備と適応

### 4-1. 入職前にしておくべき準備

**①医学知識の確認**
- 高血圧・糖尿病・心不全など慢性疾患の基礎知識
- 介護保険制度・訪問看護診療報酬について
- 感染対策（自宅での清潔・不潔）

**参考：自己学習のポイント**
- 「訪問看護師必携」（日本看護協会）
- 「在宅医療の現場から」などの実践的な教材
- YouTubeの訪問看護関連チャンネル

**②生活習慣の調整**
-
