## イントロダクション

訪問看護ステーション（以下、訪問看護事業所）の開業は、多くの看護師が描くキャリアの選択肢の一つです。在宅医療の需要拡大に伴い、新規事業所の開設ニーズは高まっています。一方、「具体的には何から始めるべき？」「必要な資格や要件は？」「資金はどの程度かかる？」といった不安を抱える方も少なくありません。

本記事では、訪問看護ステーションの開業に必要な手続き、人員配置要件、資金計画、そして成功するための準備について、実務的かつ具体的に解説します。この記事を通じて、開業への道のりを明確にし、事前に必要な対策を整える手助けができれば幸いです。

## 訪問看護ステーションの開業に必要な基本要件

### 訪問看護事業所とは何か

訪問看護ステーションは、都道府県知事の指定を受けた施設で、**居宅（自宅や高齢者施設など）を訪問し、医療的ケアと生活支援を提供する事業所**です。2024年時点で、全国には約15,000の訪問看護事業所が存在し、年間約250万人が訪問看護を利用しています。

訪問看護では、以下のようなサービスを提供します：

- **医療処置**：褥瘡（床ずれ）の処置、カテーテル管理、吸引、酸素療法など
- **療養上の世話**：入浴介助、排泄サポート、清潔ケア
- **リハビリテーション**：機能維持・向上を目的とした運動指導
- **認知機能低下予防**：メンタルヘルスサポート、生活指導
- **家族支援**：介護方法の指導、心理的支援

訪問看護事業は**介護保険および医療保険の両方から報酬を得られる**特徴があり、経営の安定性が相対的に高いため、起業の選択肢として注目されています。

### 指定を受けるための基本要件

訪問看護事業所として指定を受けるには、都道府県が定める基準を満たす必要があります。以下が主な要件です：

#### **人員配置基準**

| 職種 | 最小配置数 | 資格要件 |
|------|-----------|--------|
| 管理者 | 1名以上 | 看護師・保健師・助産師（訪問看護3年以上の実務経験） |
| 看護師・保健師 | 常勤換算で2.5名以上 | 看護師または保健師の資格 |
| 理学療法士・作業療法士・言語聴覚士 | 配置時は専従 | 各々の国家資格保持者 |
| 訪問看護職員 | 上記に含む | 看護師・保健師・准看護師など |

※常勤換算とは、複数の非常勤職員の勤務時間を合算して算出する方法です。例えば、週30時間勤務の看護師2名 = 常勤換算1.5名相当

#### **施設・設備基準**

- **事務室**：訪問看護業務に必要な記録等を整理・保管できるスペース
- **面積**：特に最小基準は定められていないが、実務上15㎡以上が目安
- **駐車場**：訪問用の車両保管スペース
- **通信機器**：利用者との連絡、ケア記録の管理が可能な体制

#### **運営上の要件**

- 365日24時間、急変時の相談対応体制
- 利用者の個人情報保護と秘密保持体制
- 感染症対策・衛生管理の整備
- 医師との連携体制

## 開業前に確認すべき関連法令と基準

### 指定基準は都道府県ごとに異なる

訪問看護の指定基準は、厚生労働省が定める**「訪問看護・訪問リハビリテーション基準」**に基づいていますが、都道府県が細部を定める裁量があります。そのため、**事業所を開設予定の都道府県の具体的な基準を確認することが重要**です。

開業検討段階で必ず以下を確認してください：

1. **都道府県の健康福祉局・医療政策課等への相談**
   - 指定申請の流れ、必要書類、審査期間
   - 人員配置の詳細基準
   - 施設・設備の具体的要件

2. **介護保険・医療保険の報酬体系の確認**
   - 利用できる加算（処置加算、ターミナルケア加算など）
   - 報酬単価の改定情報
   - 地域による単価の差異

3. **保健所への相談**
   - 衛生管理、感染症対策の要件
   - 医療廃棄物の処理基準

### 関連する主要法令

| 法令 | 主な内容 |
|------|--------|
| 介護保険法 | 介護保険サービス提供者としての指定要件 |
| 医療法 | 医療保険対応時の施設基準 |
| 個人情報保護法 | 利用者情報の管理義務 |
| 労働基準法 | 職員の雇用・労働条件に関する規制 |
| 感染症法 | 感染症発生時の報告義務 |

## 開業手続きの流れと期間

### 開業までの全体スケジュール

訪問看護ステーション開業には、**着手から指定取得まで6ヶ月～1年程度**必要です。以下に標準的なスケジュールを示します：

#### **フェーズ1：準備段階（1～2ヶ月）**

1. **都道府県への事前相談**
   - 開業予定地の決定
   - 基本的な要件の確認
   - 人員配置計画の相談

2. **事業計画の策定**
   - 事業規模（看護職員数、提供地域など）の決定
   - 初期投資額と経営予測の算出
   - 資金調達方法の検討

3. **物件選定と契約**
   - 事務所・駐車場を備えた物件探し
   - 賃貸契約の締結

#### **フェーズ2：体制整備段階（2～4ヶ月）**

1. **人員確保**
   - 管理者（看護師経験3年以上）の配置
   - 常勤看護職の採用面接・契約
   - 非常勤スタッフの登録

2. **施設・設備の準備**
   - 事務室の整備
   - 医療機器（血圧計、酸素飽和度測定器など）の購入
   - 情報システム（電子カルテ、勤務管理システム）の導入

3. **運営規程等の作成**
   - 訪問看護の提供方針と利用者受け入れ基準
   - 職員の研修・教育計画
   - 感染症対策・緊急対応マニュアル
   - 個人情報保護方針

#### **フェーズ3：申請手続き段階（2～3ヶ月）**

1. **申請書類の作成**
   - 訪問看護ステーション指定申請書
   - 事業計画書、人員配置表
   - 施設平面図、設備一覧表
   - 職員の経歴書、資格証の写し

2. **都道府県への申請**
   - 申請窓口に一括提出
   - 不備があれば修正

3. **現地調査**
   - 都道府県職員による施設調査
   - 設備の確認、運営体制の審査

4. **指定取得**
   - 都道府県知事の承認
   - 指定通知書の受領

### 申請に必要な主要書類チェックリスト

以下は都道府県ごとに異なる可能性がありますので、事前に確認してください：

- ☐ 訪問看護ステーション指定申請書（都道府県所定様式）
- ☐ 事業計画書（サービス内容、利用予定者数など）
- ☐ 管理者の経歴書、資格証の写し（看護師3年以上経験の証明）
- ☐ 常勤職員の配置表、経歴書、資格証の写し
- ☐ 法人の履歴事項全部証明書（法人設立の場合）
- ☐ 施設平面図（1/100以上の縮尺、各室の広さ表示）
- ☐ 設備・機器一覧表
- ☐ 賃貸借契約書の写し
- ☐ 運営規程（利用者の受け入れ基準、料金体系など）
- ☐ 職員研修計画書
- ☐ 感染症対策・緊急対応マニュアル
- ☐ 個人情報保護方針

## 開業に必要な資金計画と資金調達

### 初期投資の概算

訪問看護ステーション開業には、**300万円～600万円程度の初期投資**が一般的です。規模・地域・立地により幅があります。

#### **主な経費内訳**

| 項目 | 概算額（万円） | 詳細 |
|------|----------------|------|
| 事務所賃借料（3ヶ月分先払い） | 30～60 | 月額10～20万円×3ヶ月 |
| 改装・内装工事 | 50～100 | 事務室、待機スペース等の整備 |
| 医療機器・備品 | 50～80 | 血圧計、酸素計、医療用冷蔵庫など |
| 情報システム導入 | 80～150 | 電子カルテ、勤務管理、会計システム |
| 車両費（訪問用） | 100～200 | 新車購入または中古車 |
| 什器・家具 | 20～40 | デスク、椅子、ロッカーなど |
| 初期登録・指定申請費用 | 10～20 | 申請手数料、登録費用 |
| 開業前研修・コンサル費用 | 30～50 | 開業支援サービス利用時 |
| **合計** | **370～700** | |

### 月別運営経費の目安

開業後の月別経費（常勤看護師2.5名規模を想定）：

| 項目 | 月額（万円） | 備考 |
|------|------------|------|
| 人件費（給与・賞与含む） | 240～300 | 常勤2.5名、非常勤含む |
| 事務所賃借料 | 10～20 | 地域による |
| 通信費・光熱費 | 3～5 | 携帯電話、固定電話、電気ガス |
| 車両費（燃料・保守） | 5～10 | 訪問用車両の運用コスト |
| 医療材料・消耗品 | 5～10 | 衛生材料、医療用品 |
| 保険料（損害保険等） | 2～3 | 賠償責任保険など |
| 研修・教育費 | 1～2 | 職員の継続教育 |
| その他経費 | 3～5 | 事務用品、印刷費など |
| **合計** | **270～355** | |

### 資金調達の方法

#### **1. 自己資金**
- 最初の3～6ヶ月の運営資金（約80～200万円）は自己資金があると安定
- 金融機関の融資審査時の自己資金比率が重視されるため、事業費の20～30%が目安

#### **2. 金融機関からの借入**

**日本政策金融公庫（新創業融資制度）**
- 対象：新たに事業を開始する個人・法人
- 融資額：最大1,500万円
- 金利：約2.0～2.5%程度（2024年時点）
- 返済期間：最大10年（据置期間2年）

**商工中金・地域の信用金庫**
- 地域別の融資プログラムがある場合がある
- 事業計画書による審査

**信用保証協会の利用**
- 金融機関の融資に対する保証で、借入が容易になる
- 保証料が発生（融資額の0.5～2.0%程度）

#### **3. 補助金・助成金**

各都道府県や市町村で、
