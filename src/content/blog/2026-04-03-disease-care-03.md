## イントロダクション

「主治医から『自宅で様子を見るように』と言われたけど、何に気をつければいいのかわからない」「夜間に急に症状が悪くなったらどうしよう」—慢性心不全の患者さんとご家族からこうしたご相談をよく聞きます。心不全は進行が緩やかでも、ある日突然悪化して入院が必要になることがあります。しかし、**正しい自己管理の知識があれば、多くの悪化を予防できます**。この記事では、訪問看護の現場で患者さんが実践している「症状悪化を防ぐ具体的なポイント」を、医学的根拠に基づいて解説します。毎日のチェックリスト、薬の飲み方、危険な症状の見分け方までを、わかりやすくまとめました。

## 慢性心不全とは—在宅管理が必要な理由

### 慢性心不全の基礎知識と患者数

慢性心不全（きゅうせいしんふぜん：心臓が全身に血液を十分に送り出せなくなった状態が3ヶ月以上続く状態）は、日本で約100万人が罹患しているとされる重大な慢性疾患です。厚生労働省の統計によれば、高齢者の入院原因の上位3位に心不全は常に入っており、**再入院率が高い疾患の一つ**として知られています。

心不全の主な原因は、虚血性心疾患（心筋梗塞など）、高血圧、弁膜症（べんまくしょう）、不整脈などです。加齢とともに発症リスクが上がり、特に65歳以上の高齢者層で患者数が急増しています。

### なぜ在宅管理が重要なのか

心不全は**完全には治らない疾患**ですが、適切な在宅管理により：

- **再入院を60～70%減少させられる**という研究報告がある
- 生活の質（QOL）を保つことができる
- 医療費の削減にもつながる

訪問看護が導入される患者さんの約70%以上が、適切な管理を受けることで安定した在宅生活を維持できています。つまり、「正しい知識＝予防」なのです。

## 毎日実践すべき自己監理の5つのポイント

### 毎日の体重測定—最も重要な早期警告サイン

**毎日同じ時間に体重を測ること**は、心不全管理でもっとも重要な自己監理です。訪問看護師が患者さんに最初に指導することの筆頭です。

**測定のルール：**
- **時間帯：** 毎朝、排尿直後、朝食前（同じ時間帯）
- **頻度：** 毎日
- **記録方法：** 専用ノートまたはスマートフォンアプリ
- **服装：** 毎回同じくらいの重さの衣服（パジャマなど）

**目安となる体重増加：**

| 増加量 | 対応 |
|-------|------|
| 1日で1kg以上 | 即座に主治医に連絡 |
| 3日間で2kg以上 | 本日中に医師に相談 |
| 1週間で3kg以上 | 緊急対応（利尿薬調整の可能性） |

**体重が増える理由：** 心臓のポンプ機能が低下すると、体内に水分が溜まりやすくなります。この「体液貯留」が体重増加として現れるため、体重の増減は心不全の状態を最も敏感に反映します。

### 塩分制限—「控える」から「管理する」へ

「心不全患者は塩分禁止」というイメージを持つ患者さんが多いですが、実際には**完全な制限は不要**です。むしろ「適切な範囲での管理」が重要です。

**推奨塩分摂取量：**
- 目標：1日6g未満（できれば5g以下）
- 注：これは健康な人の推奨量（1日8g未満）より少ないですが、完全禁止ではありません

**実践的な塩分管理方法：**

**調理時の工夫：**
- 塩分計を活用（100円均一でも購入可能）
- 加工食品よりも新鮮な食材を優先
- 醤油・塩の代わりに酢、レモン、香辛料（わさび、からし）で味付け
- 外食時は「塩分少なめ」と注文する

**避けるべき食品：**
- 漬物、梅干し、昆布の佃煮
- ラーメン、うどん（スープに塩分集中）
- ベーコン、ハムなどの加工肉
- インスタント食品全般
- 缶詰（塩漬けのため）

**朗報：** 味噌汁や和食は工夫次第で塩分を大きく削減できます。具材を増やして味を補うテクニックを訪問看護師から学ぶことができます。

### 水分管理—「制限」の本当の意味

「心不全だから水も飲むな」と極端に解釈する患者さんもいますが、**完全な水分制限は危険**です。

**推奨される水分摂取量：**
- NYHA分類（心不全の重症度）による調整
- 一般的には：1日1,000～1,500mL程度が目安
- ただし、医師の指示によって個別対応

**実践的なポイント：**

| 場面 | 対応 |
|-----|------|
| 日中の飲水 | コップで測って、1日量を記録 |
| 食事に含まれる水分 | みそ汁、おかず、果物も含める |
| 医薬品の水分 | シロップ剤など含める |
| 寝前の飲水 | 最小限に（夜間の頻尿→安眠妨害） |

**水分管理表の例：**
```
【1日の水分摂取記録】
朝食：みそ汁1杯（200mL）+ 牛乳1杯（200mL）= 400mL
間食：お茶1杯（150mL）
昼食：ご飯 + おかず（80mL相当）= 80mL
間食：コップ1杯の水（150mL）
夕食：みそ汁1杯（200mL）= 200mL
夜間：最小限
合計：約1,030mL
```

**医師に相談するタイミング：** 日々の水分・塩分管理で改善しない浮腫（ふしゅ：むくみ）が出現した場合は、利尿薬の調整が必要な場合があります。

### 定期的な血圧・脈拍測定

体重測定と同じくらい重要なのが血圧と脈拍です。心不全患者さんの多くは高血圧や不整脈を合併しているため、これらの管理が悪化を防ぎます。

**測定のルール：**
- **頻度：** 1日2回（朝・夜）が目安
- **時間帯：** 毎回同じ時間（朝は起床直後、夜は就寝前）
- **記録方法：** 血圧手帳またはアプリ

**目安となる血圧値：**

| 血圧値 | 判定 | 対応 |
|-------|------|------|
| 140/90mmHg未満 | 良好 | 継続管理 |
| 140/90～160/100 | 軽度上昇 | 塩分・水分見直し |
| 160/100以上 | 著明上昇 | 医師に相談 |
| 90/60未満 | 低血圧 | 医師に相談（めまい、倦怠感の危険） |

**脈拍の異常を見つけるコツ：**
- **正常範囲：** 60～100回/分
- **心房細動（不整脈）:** 脈が不規則でバラバラ→**即座に医師に報告**
- **徐脈（じょみゃく）:** 50回/分以下でふらつきがある→**医師に相談**

訪問看護師は自動血圧計の使い方も指導してくれるため、測定方法に不安があれば遠慮なく質問してください。

### 症状の日記記録—変化を見逃さない

体重・血圧だけでなく、**症状の細かい変化**を記録することも重要です。医師も訪問看護師も、患者さんの記録を診療の参考にします。

**記録すべき項目：**

**毎日記録する項目：**
- 呼吸の苦しさ（0=なし～10=非常に苦しい、数値で）
- むくみの有無と程度（足首、顔など）
- 夜間の目覚め回数
- 疲労感の程度
- 睡眠時間と質

**異常があれば記録する項目：**
- 咳が出ている（乾いた咳か痰が出るか）
- 胸痛や違和感
- めまいやふらつき
- 動悸（心臓がドキドキしている感覚）
- 意識の変化（ぼんやりしている）

**記録テンプレート例：**

```
【2026年4月3日（木）】
・体重：62.5kg（前日比+0.5kg）
・血圧：朝138/85  夜145/88
・呼吸の苦しさ：3（階段では5に上昇）
・むくみ：足首に軽度あり（昨日と同じ）
・夜間排尿：2回（普段より多い）
・咳：なし
・特記事項：昨日より疲れやすい感じ
```

この情報があれば、訪問看護師や医師は「今後の悪化リスク」をより正確に判断できます。

## 危険な症状と対応方法—この症状が出たら医療機関へ

### すぐに医師・救急に連絡すべき危険な症状

以下の症状が出現した場合は、**躊躇なく医師に連絡、または119に電話して救急車を呼んでください。**

**最優先で対応が必要な症状：**

| 症状 | 説明 | 対応 |
|-----|------|------|
| **急性の息切れ（呼吸困難）** | 安静にしていても呼吸が苦しい、息が続かない | **119に電話** |
| **夜間発作性呼吸困難** | 寝ている途中に突然息ができなくなり、起き上がる | **119に電話** |
| **ピンク色の泡状痰** | 肺に液体が溜まっている兆候（急性肺水腫） | **119に電話** |
| **胸痛** | 締め付けられるような胸部痛、特に左肩への放散痛 | **119に電話** |
| **失神または意識喪失** | 気を失う、または数秒間の意識消失 | **119に電話** |
| **著明な動悸** | 心臓が激しく鼓動し、めまいを伴う | **医師に即連絡** |
| **尿が出ない（24時間以上）** | 腎臓機能低下の兆候 | **医師に即連絡** |

### 医師に相談が必要な症状（緊急ではないが対応が必要）

**本日中に医師に相談すべき症状：**
- 3日間で2kg以上の体重増加
- むくみが急に増した（靴が入らない、指輪が外れなくなるなど）
- 新たに咳が出始めた、または咳が増えた
- 夜間排尿が3回以上に増加した（本来は1～2回の場合）
- 食欲がなくなった
- 疲労感が強い
- 軽い運動で以前より息切れするようになった

### 誤解しやすい症状の見分け方

**「これは心不全が悪化したのかな？」と疑いやすい症状：**

**足のむくみ：**
- 心不全の可能性：左右両足対称にむくむ、体重増加を伴う
- 他の原因の可能性：片足だけ、静脈瘤がある、最近歩き回った

**咳：**
- 心不全の可能性：乾いた咳、夜間に強い、活動時に増す
- 他の原因の可能性：痰が出る、
