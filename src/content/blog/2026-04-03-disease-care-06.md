## イントロダクション

がん患者が自宅での療養を選択する際、最も心配されることの一つが「痛みのコントロールが本当にできるのか」という不安です。「病院にいないと痛みが管理できないのではないか」「緩和ケアは終末期だけのものでは」——こうした誤解や懸念を持つ患者さんと家族は少なくありません。

しかし実は、**適切な訪問看護体制と医学的知識があれば、自宅での疼痛管理は十分に可能です**。むしろ、自分のペースで療養でき、家族に見守られながら過ごせる在宅療養は、多くのがん患者にとって生活の質を大きく向上させます。

本記事では、訪問看護師の実践経験に基づいた疼痛管理の方法論、緩和ケアの具体的な実施手順、そして家族が知っておくべき知識をまとめました。この記事を通じて、**在宅でも質の高い痛みのコントロールが実現できること**を理解していただきたいと思います。

## がん在宅療養における疼痛管理の現状

### 日本におけるがん患者の在宅療養選択率

日本のがん患者の在宅療養は年々増加しています。厚生労働省の「患者調査」（2023年）によると、がん患者全体の約30～40%が在宅療養を希望していますが、実際の在宅療養率は地域によって大きなばらつきがあります。都市部では20～30%、地方では10～15%程度が現状です。

在宅療養を選択しない理由の上位3つは：
- **疼痛管理への不安**（56%）
- 家族の介護負担への懸念（48%）
- 医療体制の不足（42%）

つまり、疼痛管理の質を向上させることで、在宅療養の選択肢を広げられる余地が大きいのです。

### なぜ在宅での疼痛管理は可能なのか

在宅での疼痛管理が可能な理由は以下の通りです：

| 要素 | 在宅療養の利点 |
|------|----------------|
| **薬物療法** | 強オピオイド（モルヒネ等）の処方・投与が可能 |
| **投与経路の多様性** | 経口薬だけでなく、貼付剤・注射などが使用可能 |
| **アセスメント** | 患者の生活環境を直接観察でき、個別対応が容易 |
| **継続性** | 同じ訪問看護師による継続的な評価が可能 |
| **心理的環境** | 慣れた自宅環境で心理的ストレスが低い |

## 訪問看護師による疼痛アセスメントの実践

### 初回訪問時のアセスメント項目

訪問看護師が初回訪問で確認すべき疼痛関連の項目を、チェックリストで示します：

**初回訪問・疼痛アセスメントチェックリスト**

```
□ 現在の痛みの有無と程度
  ├─ 数値評価スケール（NRS: 0～10）での評価
  ├─ VAS（Visual Analogue Scale）での視覚化
  └─ 言語的記述（軽い/中程度/強い/耐えがたい）

□ 痛みの特性
  ├─ 痛みの部位（正確な位置を図示）
  ├─ 痛みの性質（鋭い/鈍い/焼ける/しびれ感）
  ├─ 痛みの起伏パターン（持続的/間欠的）
  └─ 痛みの発症時期と経過

□ 現在の薬物療法
  ├─ 処方内容（薬剤名・用量・服用回数）
  ├─ 服用状況（実際に飲めているか）
  ├─ 副作用の有無と程度
  └─ 効果判定（十分か/不十分か）

□ 日常生活への影響
  ├─ 睡眠障害の有無
  ├─ 食事摂取への影響
  ├─ 身体活動の制限
  └─ 心理状態（不安/抑うつ）

□ 既往歴と背景
  ├─ がんの種類と進行度
  ├─ 既施行治療（手術/化学療法/放射線治療）
  ├─ 他疾患の有無
  └─ アレルギー歴

□ 生活環境と家族背景
  ├─ 同居家族の状況
  ├─ 介護者の心身状態
  ├─ 自宅での福祉用具の準備状況
  └─ 社会的サポート体制
```

### 疼痛スケールの選択と使用方法

異なるスケールの選択基準：

**NRS（数値評価スケール）**
- **推奨対象**：意思疎通が可能な患者
- **方法**：「0（痛みなし）～10（これ以上ない痛み）」の数字を選ばせる
- **頻度**：初回・毎訪問時、および治療変更時
- **利点**：シンプルで繰り返し評価しやすい

**VAS（視覚的アナログ・スケール）**
- **推奨対象**：より詳細な評価が必要な場合
- **方法**：10cm の直線上に患者が自分の痛みの位置にマークする
- **利点**：より感度が高く、わずかな変化を捉えやすい

**フェイススケール（顔スケール）**
- **推奨対象**：高齢者や認知機能が低下している患者
- **方法**：異なる表情の顔から該当するものを選ぶ
- **利点**：言語理解が不要で理解しやすい

## がん疼痛に対する薬物療法の基本

### WHO3段階鎮痛ラダーの理解と応用

世界保健機関（WHO）が推奨する**WHO鎮痛ラダー**は、がん疼痛管理の国際標準です。訪問看護師はこの体系を理解した上で、医師との連携を密にする必要があります。

```
【WHO3段階鎮痛ラダー】

第1段階（軽度の痛み：NRS 1-3）
├─ 第一選択薬：非オピオイド鎮痛薬
│  ├─ アセトアミノフェン
│  ├─ NSAIDs（ロキソニン等）
│  └─ 随伴症状があれば随伴薬を併用
└─ 用量調整：市販薬並みから開始することが多い

  ↓（痛みが軽減しない場合、または増悪した場合）

第2段階（中度の痛み：NRS 4-6）
├─ 第一選択薬：弱オピオイド ± 非オピオイド鎮痛薬
│  ├─ コデイン
│  ├─ トラマドール
│  ├─ タペンタドール
│  └─ 併用：アセトアミノフェンやNSAIDs
└─ 特徴：非オピオイドとの併用で相乗効果が期待できる

  ↓（痛みが軽減しない場合、または強度が増す場合）

第3段階（強度の痛み：NRS 7-10）
├─ 第一選択薬：強オピオイド ± 非オピオイド鎮痛薬
│  ├─ モルヒネ（経口/注射/貼付）
│  ├─ オキシコドン（経口/注射）
│  ├─ フェンタニル（貼付/注射）
│  ├─ ヒドロモルフォン
│  └─ 随伴症状に応じた随伴薬の併用
└─ 特徴：段階的に用量を増やす「滴定」が重要

【全段階共通】
├─ 随伴症状の管理
│  ├─ 便秘対策（下剤）
│  ├─ 悪心・嘔吐対策（制吐薬）
│  ├─ 不安・抑うつ対策（抗不安薬/抗うつ薬）
│  └─ せん妄対策（抗精神病薬）
└─ 定期的なアセスメントと調整

```

### 強オピオイド（モルヒネ）の基本知識

訪問看護師が押さえておくべき強オピオイドの知識：

**モルヒネの投与経路と特徴**

| 投与経路 | 製品例 | 発現時間 | 持続時間 | 訪問看護での利点 |
|---------|--------|---------|---------|----------------|
| **経口（速放性）** | 散剤・錠剤 | 30-60分 | 3-4時間 | 使用が簡単、調整が容易 |
| **経口（徐放性）** | MS コンチン等 | 1-2時間 | 8-12時間 | 服用回数が少ない、アドヒアランス向上 |
| **皮膚貼付** | フェントステッチ等 | 12-24時間 | 72時間 | 服用負担がない、計画的な投与 |
| **坐剤** | アンペック等 | 30-60分 | 4-6時間 | 経口困難時に有効 |
| **皮下注射** | オピオイド注射液 | 数分 | 数時間 | 急激な悪化時の速効性 |
| **静脈注射** | 点滴ポンプ使用 | 直後 | 持続的 | 安定した疼痛管理、難治性痛み対応 |

**モルヒネの用量調整の原則（「滴定」）**

```
開始用量から段階的に増量する方法：

① 開始用量の設定
   └─ 患者の既往歴・肝腎機能・年齢を勘案して決定
   └─ 通常は1日経口モルヒネ換算30-60mg から開始

② 定期的な評価
   └─ 毎訪問時にNRSで疼痛評価
   └─ 副作用（便秘・悪心等）の確認

③ 用量の段階的増量
   └─ 効果不十分な場合、1日用量の25-50%増量が目安
   └─ 3-7日間隔で評価・調整
   └─ 患者の耐容性を常に確認

④ 投与経路の検討
   └─ 経口が困難になった場合は坐剤・注射への変更を検討
   └─ 医師の指示に基づき実施
```

### 突出痛（ブレークスルーペイン）への対応

定期的に投与されている薬によってもコントロールされない、突然の激しい痛みを「突出痛」または「ブレークスルーペイン」と呼びます。

**突出痛の特徴と対応**

```
【突出痛の特徴】
- 予測不可能に発生（または予測可能に発生）
- 通常、数分～1時間で強度に達する
- 継続時間は数分～数時間（通常は1時間以内）
- 日に数回～十数回発生することもある
- がん患者の50-60%が経験

【訪問看護師の対応】
1. 突出痛の詳細な記録
   ├─ 発生時刻・頻度・持続時間
   ├─ 強度（NRS）
   ├─ 誘因の有無（動き/排便/感情など）
   └─ 治療の効果

2. 医師への報告
   ├─ 記録した情報を基に医師に相談
   ├─ 背景痛の管理が不十分でないか検討
   └─ 突出痛用の追加薬剤処方を検討

3. 突出痛用薬剤
   ├─ 速放性オピオイド（経口液・散剤）
   ├─ 口腔粘膜吸収製剤（素早い効果）
   └─ 肛門坐剤（嚥下困難時）

4. 非薬物的対応
   ├─ 痛みの誘因を除去（姿勢変換等）
   ├─ リラクセーション
   ├─ 心理的サポート
   └─ 環境調整
```

## がん緩和ケアの実践的アプローチ

###
