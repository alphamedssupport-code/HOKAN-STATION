## イントロダクション

脊髄損傷は、受傷部位に応じて様々な合併症のリスクを伴う疾患です。特に自宅での療養が長期化する場合、**自律神経障害（汗の異常、血圧変動）**と**褥瘡（床ずれ）**は生活の質を大きく低下させる主要な課題となります。在宅で患者さんを支える家族やケアラーの中には「本当にこのケアで大丈夫だろうか」「急変に対応できるだろうか」といった不安を抱える方が多いのが実情です。本記事では、訪問看護師が現場で実践している脊髄損傷患者の自律神経障害管理と褥瘡予防・対応策を、具体的な手順とチェックリスト、根拠となるガイドラインに基づいて解説します。これにより、あなたが安心して在宅ケアを実施し、患者さんの合併症発症を予防できるようサポートします。

## 脊髄損傷と在宅ケアの基礎知識

### 脊髄損傷の分類と在宅生活への影響

脊髄損傷（せきずいそんしょう）は、脊椎の損傷に伴って脊髄が圧迫・切断される状態を指します。損傷部位と程度によって、以下のように機能障害の範囲が異なります。

**損傷部位による分類：**

| 損傷部位 | 機能障害の範囲 | 在宅ケアでの主な課題 |
|---------|-------------|-------------------|
| 頚髄（首） | 四肢麻痺（全身） | 呼吸補助の必要性、自動車運転困難 |
| 胸髄（胸部） | 下肢麻痺、体幹機能低下 | 褥瘡リスク高い、排尿管理 |
| 腰髄（腰部） | 下肢麻痺 | 歩行支援、排泄管理 |

受傷後は、脳からの信号が脊髄で遮断されるため、損傷部位より下の身体は指令を受け取れなくなります。これにより、運動機能喪失だけでなく、**自律神経の調節機能も大きく影響を受ける**ことになります。特に胸髄以上の損傷では、自律神経障害（後述）のリスクが格段に上がります。

### 訪問看護が関わる主要な在宅ケア内容

脊髄損傷患者の在宅ケアにおいて、訪問看護師は以下のような医療管理を担います：

- **排尿・排便管理**：導尿、便管理指導
- **スキンケア・褥瘡管理**：予防および既発生褥瘡の処置
- **自律神経障害への観察と対応**：血圧変動、汗の異常、脈拍変化の監視
- **合併症予防教育**：肺炎、尿路感染、血栓症などの予防指導
- **日常生活動作（ADL）支援**：入浴、更衣、移動の援助

本記事では、特に**自律神経障害**と**褥瘡管理**に焦点を当てます。これらは在宅療養期間中に頻繁に直面し、対応に工夫が必要な課題だからです。

## 脊髄損傷に伴う自律神経障害の理解と管理

### 自律神経障害の発症メカニズムと症状

脊髄損傷に伴う自律神経障害（dysreflexia）は、損傷部位より下の身体からの知覚情報が脳に到達せず、脊髄レベルで反射的に過剰な自律神経反応が起こる状態です。**特に胸髄第6レベル以上の損傷で発症リスクが高い**とされています。

**自律神経障害の典型的な症状（発作時）：**

- 血圧の急上昇（通常時より20mmHg以上の上昇）
- 激しい頭痛
- 顔面・頸部の紅潮、発汗
- 徐脈（脈拍低下）
- 視覚障害、眼痛
- 胸痛
- 不安感

**日本脊髄障害医学会のガイドラインによると、自律神経障害は命に関わる緊急事態であり、迅速な対応が必須です**。脳出血や急性冠症候群に至る可能性があるため、重視すべき合併症の筆頭です。

### 自律神経障害の主な誘発因子と予防戦略

在宅ケアにおいて、発作を未然に防ぐことが最優先です。以下は訪問看護師が把握すべき主要な誘発因子です：

**最も頻繁な誘発因子（％）：**

1. **尿排液（膀胱充満・尿路感染）**：30-40%
2. **便の停滞・腸刺激**：25-35%
3. **皮膚刺激（褥瘡、タイトな衣類、温度変化）**：15-25%
4. **月経痛、射精関連**：10-15%
5. **不適切な導尿チューブ操作**：5-10%

#### 予防戦略① 尿路系の管理

**毎日のチェック項目：**

- 導尿の実施時間を一定にする（通常4-6時間間隔）
- 尿の色・濁り・異臭の有無を記録する
- 尿量が通常より明らかに少ない場合は医師に報告
- 尿路感染症の兆候：発熱、尿の濁り、異臭、血尿

**予防のための実践的手順：**

1. 導尿前に手指衛生を徹底する
2. 清潔な器具を使用し、カテーテル挿入時の外傷を最小限に
3. 十分な水分摂取（1日1500-2000mL程度）を指導
4. 定期的な尿検査（月1-2回程度）で無症候性菌尿を早期発見
5. 自動排尿脊髄損傷患者については、排尿に関する指導書を家族と共有

#### 予防戦略② 排便管理と腸刺激の回避

脊髄損傷患者は便秘になりやすく、硬便による腸刺激は自律神経障害の強力な誘発因子となります。

**排便管理の基本スケジュール：**

- 毎日ほぼ同じ時間に排便を促す習慣をつける
- 食後30分以内に便意を感じやすいため、朝食後が理想的
- 便秘予防：線維質食の摂取（野菜、全粒穀物）、十分な水分
- 必要に応じて軟便剤（酸化マグネシウムなど）を医師の指示下で使用

**便刺激による誘発を避けるため：**

- 浣腸・摘便は医療的に必要な場合のみ、医師・看護師指導下で実施
- 強力な腸刺激を避ける方法として、定期的な排便習慣形成を優先
- 便が硬い場合は、担当医に報告し便軟化剤の調整を依頼

#### 予防戦略③ 皮膚の刺激回避

褥瘡や皮膚の小さな刺激も誘発因子となり得ます。

- タイトな衣類、靴、装具を避ける
- 毎日の身体観察で皮膚異常（発赤、傷）を早期発見
- 褥瘡予防と管理（次章で詳述）
- 室温を20-24℃程度に保つ（温度変化の予防）

### 自律神経障害の急性対応プロトコル

万が一、患者さんが自律神経障害の症状を示した場合、以下の対応を直ちに実施します。**これは医療的な緊急事態です**。

**対応ステップ（直ちに実施）：**

1. **患者を座位又は半座位に保つ**（血圧低下時の脳血流を維持）
2. **血圧・脈拍を測定し、時間と数値を記録する**
3. **誘発因子を特定し除去する：**
   - 導尿チューブが詰まっていないか確認→詰まっていれば新しいカテーテルに交換
   - 膀胱充満がないか確認→必要に応じて導尿
   - 衣類がタイトでないか確認→緩める
   - 便の停滞がないか腹部を観察
4. **医師への直ちの連絡・報告**（症状、血圧、採取した対応内容）
5. **症状が軽減するまで患者の様子を継続監視**

**このプロトコルについては、事前に患者さん・ご家族・担当医で繰り返し確認し、書面で保管してください。**

## 褥瘡（床ずれ）の予防と段階別管理

### 脊髄損傷患者が褥瘡を発症しやすい理由

脊髄損傷患者は、以下の複数の要因により**褥瘡発症リスクが健常者に比べて著しく高い**です：

**褥瘡発症のメカニズム：**

| リスク要因 | 脊髄損傷患者での該当状況 |
|----------|----------------------|
| 圧迫 | 動く能力がない、また感覚がないため自分で寝返りできない |
| 感覚障害 | 圧迫感を感じないため、皮膚の警告信号に気づけない |
| 動きの制限 | 筋力喪失で体位変換ができない |
| 湿潤環境 | 失禁、発汗、皮膚折り畳み部の蒸れ |
| 栄養不良 | 長期臥床で筋肉量低下、蛋白質不足 |
| 多尿・頻尿 | 尿失禁による皮膚浸軟 |

**統計データ：日本褥瘡学会の報告では、脊髄損傷患者の褥瘡発症率は一般病院患者の3-5倍に達します**。特に初期の在宅移行期（受傷後6ヶ月-2年）での発症が多いため、この時期の対策が極めて重要です。

### 褥瘡予防：日常ケアチェックリスト

効果的な褥瘡予防は「毎日の地道な観察とケア」に尽きます。以下のチェックリストを壁に貼るか、ご家族のスマートフォンにリマインダーを設定して、毎日実行してください。

**褥瘡予防チェックリスト（毎日実施）：**

- [ ] **朝の身体観察**：全身の皮膚を観察し、発赤・こすれ痕・傷がないか確認（特に骨突出部：仙骨、坐骨、踵、肘、肩甲骨）
- [ ] **2時間ごとの体位変換**：仰臥位→側臥位→反対側臥位→仰臥位 のサイクル、または医師の指示に従う
- [ ] **皮膚の湿度管理**：失禁時は直ちにおむつ・下着を交換し、皮膚を清潔・乾燥に保つ
- [ ] **栄養・水分摂取の確認**：1日蛋白質60g以上、水分1500-2000mL程度
- [ ] **除圧用具の装着確認**：褥瘡予防クッション（エアマットレス、ジェルクッションなど）が正常に機能しているか
- [ ] **衣類・シーツの確認**：シーツのしわ、タイトなバンド、硬い縫い目がないか
- [ ] **排泄管理**：便秘予防と尿失禁対策で皮膚の蒸れを最小化
- [ ] **爪・髪の毛の確認**：患者の身体に挟まり込んでいないか

**重要：このチェックリストに加え、訪問看護師による月1-2回の評価と指導を受けることを強く推奨します。**

### 褥瘡ステージ別対応と処置方法

万が一褥瘡が発生してしまった場合、その進行
