## イントロダクション

精神疾患を持つご家族が退院し、自宅での療養生活が始まる――そのとき、不安や戸惑いを感じるのは自然なことです。「毎日どのように接すればいい？」「症状が悪化したときは？」「投薬管理はどうする？」といった疑問が尽きません。統合失調症やうつ病などの精神疾患は、身体疾患と異なり、目に見えない苦しさがあり、対応方法も複雑です。

しかし、適切な医学的知識、訪問看護サービスの活用、そして家族のサポートがあれば、自宅での安定した生活は十分可能です。本記事では、精神疾患を持つ患者さんへの訪問看護について、症状の理解から日常的なケアのポイント、支援体制の構築まで、実践的な情報をお届けします。医療従事者の視点と家族の視点の両方から、自宅療養を成功させるための具体的なアプローチを解説します。

## 精神疾患患者の在宅療養が増えている背景

### 精神保健福祉施策の転換とトレンド

日本の精神保健医療政策は大きく変わりました。かつて精神疾患は「入院治療」が中心でしたが、2004年に厚生労働省が打ち出した「精神保健福祉の改革ビジョン」以降、「地域でのケア」へのシフトが加速しています。

2022年時点で、精神科医療機関の入院患者数は約27万人ですが、通院患者数は約320万人と、圧倒的多数が地域で生活しています。この背景には以下の要因があります：

- **脱施設化・脱入院化政策**：長期入院による弊害（生活機能の低下、社会からの孤立）を認識し、可能な限り在宅・地域での生活を支援する方針
- **薬物療法の進化**：新しい向精神薬（特に非定型抗精神病薬）の開発により、副作用が減少し、外来管理が容易に
- **地域包括ケアシステムの構築**：訪問看護、訪問診療、居宅介護などの在宅サービスの充実

これにより、精神疾患患者が自宅で安定した生活を送るための環境が整備されつつあります。

### 訪問看護の役割の重要性

精神疾患の訪問看護は、単なる「医学的な観察」に留まりません。以下の重要な役割があります：

- **薬物療法の支援**：服薬管理、副作用の早期発見、医師への報告
- **症状の自己管理支援**：症状の初期兆候の認識、対処技能の習得
- **生活技能の向上**：食事、睡眠、衛生管理、日中活動などの日常生活サポート
- **社会参加の促進**：就労や通院の継続、人間関係の構築支援
- **家族教育・支援**：家族の負担軽減、対応方法の指導、相談窓口の提供

訪問看護師は、患者さんの自宅という「その人らしい生活の場」で、医療と生活支援を統合的に提供できる存在です。

## 統合失調症の特徴と訪問看護のポイント

### 統合失調症の基本理解

統合失調症は、脳の神経伝達物質（特にドーパミン）のバランスが崩れることで生じる精神疾患です。**有病率は約1%**で、決して稀な疾患ではありません。発症は15～35歳の若年層が多く、人生の重要な段階での発症となります。

**主な症状は以下の2つのグループに分けられます：**

| 症状の分類 | 具体例 | 患者さんの体験 |
|---------|------|------------|
| **陽性症状**（症状が過剰に出現） | 幻聴、妄想、思考障害 | 「誰かが監視している」「声が聞こえる」と確信している |
| **陰性症状**（症状が減弱） | 感情の平坦化、意欲低下、社会的引きこもり | やる気が出ない、誰とも話したくない感覚が続く |

**さらに認知機能の低下**も重要で、注意集中、記憶、実行機能が障害されることがあります。

### 訪問看護での初期アセスメント

訪問開始時には、以下のポイントを重点的に評価します：

**1. 現在の症状レベルの把握**
- 幻聴・妄想の有無と内容
- 日中の活動レベルと睡眠パターン
- 衛生管理（入浴、着替え）の状況
- 食事摂取量・栄養状態

**2. 薬物療法への適応状況**
- 現在の処方薬と服薬状況
- 副作用の有無（錐体外路症状、体重増加、口渇など）
- 薬物療法への理解度と受け入れ

**3. 家族および社会関係**
- 同居家族と患者さんの関係性
- 家族の対応方法や負担状況
- 孤立の程度、友人関係の有無

**4. 危険行為・自傷他害のリスク**
- 幻聴の内容に危険性がないか
- 妄想に基づいた行動の可能性
- 自殺念慮の有無

### 薬物療法の支援と副作用対応

統合失調症の治療の中心は**抗精神病薬**です。以下のポイントで支援します：

**服薬管理の実装的アプローチ：**

```
【服薬管理チェックリスト】
□ 1回分ずつ小分けにした薬を用意する（7日分のピルケースなど）
□ 飲んだ日時をカレンダーに記録させる
□ 飲み忘れがあった場合の対応を事前に医師と相談
□ 副作用が出た場合の連絡先を明確にする
□ 定期的に医師の指示通り服用できているか確認
□ 薬の効果を実感できない場合は、自己判断で中断しないよう促す
```

**主な副作用と対応方法：**

| 副作用の種類 | 症状 | 訪問看護での対応 |
|-----------|------|------------|
| **錐体外路症状** | 筋肉の硬直、手の震え、落ち着きのなさ | 医師に報告、抗パーキンソン薬の処方を相談 |
| **体重増加・高血糖** | 特に第2世代抗精神病薬で顕著 | 栄養指導、運動促進、定期的な体重測定 |
| **高プロラクチン血症** | 女性の生理不順、乳汁分泌、性機能障害 | 医師に相談、薬剤変更の検討 |
| **起立性低血圧** | 立ちくらみ、転倒リスク | 急激な体位変化を避けるよう指導 |
| **口渇・便秘** | 水分摂取、便秘症状 | 十分な水分補給、食物繊維摂取の促進 |

### 日常生活支援と生活技能訓練

陰性症状が強い患者さんの場合、やる気の欠如により日常生活が崩れやすくなります。

**具体的な支援内容：**

**1. 生活リズムの確立**
- 毎日同じ時間に起床する習慣
- 朝日を浴びることの重要性を説明
- 規則的な食事時間の設定

**2. 衛生管理の支援**
- 入浴が困難な場合は、清拭やシャンプー介助
- 衣類の着替え習慣づけ
- 歯磨きなど口腔衛生

**3. 食事・栄養管理**
- 体重増加が予測される薬物使用時は特に重要
- 献立の工夫（高カロリー飲料を避けるなど）
- 患者さん自身が食事準備できるようにサポート

**4. 認知機能リハビリテーション**
- 記憶力低下への対応：写真や図を使った記録、スマートフォンのリマインダー機能の活用
- 注意集中の促進：手工芸、読書、パズルなど段階的な活動

### 症状悪化の初期兆候の察知と対応

**統合失調症の再発予兆**（訪問看護師が重視する観察項目）：

```
【再発兆候チェックリスト】（再発の1～2週間前に出現することが多い）
□ 睡眠時間が著しく減少（通常より2時間以上少ない）
□ 落ち着きがなくなり、そわそわしている
□ 話題が次々と変わり、思考が散乱
□ テレビやラジオの内容が自分に関連していると言い出す
□ 一人で喃語（ぶつぶつ独り言）を言う頻度が増加
□ 食事摂取が著しく減少
□ 入浴などの身辺整理をしなくなった
□ 外出を極度に拒否
□ 不安や恐怖の訴えが増加
```

**初期兆候発見時のアクション：**
1. 詳細な様子を記録（いつから、どの程度など）
2. 直ちに訪問医師・精神科医に報告
3. 家族に情報共有し、見守り体制を強化
4. 必要に応じて外来受診や入院の検討

## うつ病の特徴と訪問看護のポイント

### うつ病の基本理解と患者さんの苦しみ

うつ病は、**気分障害の中で最も一般的**な疾患で、生涯有病率は約5～10%です。単なる「気分が落ち込んでいる状態」ではなく、脳神経生物学的な疾患です。セロトニンやノルアドレナリンなどの神経伝達物質が不足することで生じます。

**うつ病の代表的な症状：**

| 症状のカテゴリ | 具体的な症状 |
|----------|---------|
| **気分症状** | 強い抑うつ気分、興味・喜びの喪失、無価値感、罪悪感 |
| **身体症状** | 不眠（早朝覚醒など）、食欲不振、疲労感、体がだるい |
| **認知症状** | 集中困難、決断困難、記憶障害、思考の遅延 |
| **行動症状** | 外出困難、社会活動の回避、自傷行為 |
| **自殺念慮** | 「死にたい」という強い願望（医学的に重要） |

### 危険度評価と自殺予防

うつ病患者さんの自殺リスク評価は、訪問看護の**最重要課題**です。わが国の自殺者数は年間約20,000人で、そのうち約60%が抑うつ状態にあると推定されています。

**訪問看護での自殺リスク評価ツール（簡易版）：**

```
【自殺リスク評価スコア】
次の項目について、該当するものを数える

高リスク因子：
□ 自殺念慮の明確な存在（「死にたい」と何度も言及）
□ 自殺計画の具体性（方法・時期が具体的）
□ 以前の自殺未遂歴
□ 近親者の自殺歴
□ 重症度（日中の活動がほぼ皆無）
□ 衝動性が高い（怒りやすい、落ち着かない）
□ 社会的孤立（誰とも接触していない）
□ アルコール乱用または薬物乱用
□ 身体疾患の併存（慢性痛など）

スコア：0-2点 → 低リスク / 3-5点 → 中リスク / 6点以上 → 高リスク
→ 高リスクの場合は直ちに医師・精神保健福祉センターに報告
```

**自殺危機の初期兆候**：
- 突然、気分や行動が改善（遺書を書いた、決意が固まったことによる）
- 大切なものを処分し始める
- 別れの言葉
