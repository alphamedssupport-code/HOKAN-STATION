## はじめに

「親が骨粗鬆症と診断されてから、転んでしまわないか心配で眠れない」「退院後、自宅での過ごし方がわからない」。こうした不安は、骨粗鬆症や骨折を経験した高齢者とその家族に共通する課題です。実は、日本の骨粗鬆症患者数は約1,280万人にのぼり、そのうち高齢者の転倒による骨折は寝たきりに至る大きなリスク因子となっています。

しかし、自宅で適切なケアと予防対策を実施すれば、生活の質を保ちながら安全に過ごすことは十分可能です。本記事では、訪問看護師の現場経験に基づき、骨粗鬆症・骨折後の在宅ケアにおける**転倒予防策**、**疼痛管理の方法**、**日常生活支援のポイント**を、具体的な手順とチェックリストでご説明します。医学的根拠に基づき、家族や本人が実践できるケアをご紹介しますので、退院後の生活に不安を感じている方はぜひご活用ください。

## 骨粗鬆症と骨折のリスク理解

### 骨粗鬆症とは

骨粗鬆症（こつそしょうしょう）とは、骨の密度が低下し、骨が脆くなる疾患です。骨は常に古い骨が破壊され、新しい骨が作られる「骨のリモデリング」というプロセスを繰り返していますが、加齢やホルモン変化により、骨密度が減少します。

**主な特徴：**
- 症状がほとんどない（「無症状の病気」と呼ばれることも）
- 転倒などの軽い衝撃で骨折しやすくなる
- 特に女性は閉経後5～10年で急速に骨密度が低下
- 日本の65歳以上の女性の約3人に1人が罹患

厚生労働省の調査によると、骨粗鬆症による骨折は年間約100万件発生しており、うち約50万件が大腿骨近位部骨折（太ももの付け根の骨折）で、これは寝たきりの主要な原因の一つです。

### 在宅での高齢者の転倒リスク

在宅環境での転倒は、病院内よりもはるかに多く発生します。高齢者の約30～40%が年に1回以上転倒を経験しており、転倒した人の約50%が再度転倒するという報告もあります。

**自宅での主な転倒原因：**
- 床の段差や敷物のつまずき（約40%）
- トイレやお風呂での滑り（約20%）
- 階段の転落（約15%）
- 脚の筋力低下と視力低下の複合（約25%）

特に骨粗鬆症のある方が転倒すると、健康な方よりも骨折リスクが10倍以上高まります。転倒予防は、在宅での骨粗鬆症ケアにおいて**最優先課題**と言えます。

## 転倒予防のための環境整備

### 自宅環境の危険箇所チェック

転倒予防の第一歩は、自宅環境の徹底した安全評価です。訪問看護師や理学療法士と協力して、以下のチェックリストで危険箇所を把握しましょう。

**自宅転倒リスク評価チェックリスト**

| 場所 | チェック項目 | 対応策 |
|------|-----------|--------|
| **玄関** | 段差（通常5～8cm）の有無 | スロープ設置またはベッチ（段差解消ブロック） |
| | 靴の脱ぎ履きバランス | 安定した椅子を置く |
| | 照明不足 | LED照明の増設 |
| **廊下・居間** | 敷物やカーペット | 撤去またはテープで固定 |
| | 電源コードの露出 | クリップで壁に固定 |
| | 家具の配置 | 移動路を広げ、整理整頓 |
| **トイレ** | 床の濡れやすさ | 速乾性マットの設置、滑り止め加工 |
| | 便座の高さ | 便座リフター（クッション）で高さ調整 |
| | 手すりの欠落 | L字形またはI字形手すりの設置 |
| **浴室** | 浴槽からの出入り | バスボード・シャワーチェア導入 |
| | 床の滑りやすさ | 浴室内滑り止めマット敷設 |
| | 照度不足 | 防水LED照明の追加 |
| **寝室** | ベッドの高さ | 床から40～45cm（両足が床に着く高さ） |
| | ベッド周囲の障害物 | 移動路を1m以上確保 |
| | 夜間の照明 | フットライトやセンサー式照明 |
| **階段** | 手すりの有無 | 両側に手すり設置が理想的 |
| | ステップの高さの不均一 | 滑り止めテープで各段をマーク |
| | 照明 | 階段全体を明るく照らす |

### 具体的な転倒予防用具の活用

自宅改修には相応の費用がかかりますが、介護保険制度を活用できる場合があります。以下の用具の活用を検討してください。

**介護保険の住宅改修対象用具（自己負担1割～3割）**
- 手すり取付け：最大20万円
- 段差解消：最大20万円
- トイレ便座リフター：最大20万円
- 浴室改修：最大20万円
- その他床材変更等：最大20万円
（合計上限額：自己負担1割で2万円程度の目安）

ケアマネジャーに相談すれば、リハビリ専門職による評価を受け、最適な改修計画を立てられます。

### 照明環境の最適化

多くの転倒は薄暗い環境で発生します。高齢者は若年者よりも多くの光量が必要です（目安として3倍程度）。

**照明改善のポイント：**
- 寝室：フットライト（夜間のトイレ時に足元を照らす）の設置
- トイレ：24時間点灯型センサーライトの導入
- 廊下：人感センサー付きLED照明で自動点灯
- 階段：各段の縁を異なる色でマーク（視覚的コントラスト）
- リビング：昼間は南向き窓からの自然光を活用

## 骨粗鬆症の医学的管理と栄養

### 骨粗鬆症の医学的治療

骨粗鬆症の治療は、主治医の指示に基づき進められます。在宅での医学的管理を理解することで、服薬継続率の向上につながります。

**主な骨粗鬆症治療薬**

| 薬剤分類 | 代表例 | 特徴 | 注意点 |
|---------|--------|------|--------|
| **ビスホスホネート製剤** | アレンドロン酸、リセドロン酸 | 骨吸収を抑制する（第一選択薬） | 食道炎を避けるため空腹時に、30分後から飲食可 |
| **SERM** | ラロキシフェン | 女性特有のホルモン変化に対応 | 血栓症のリスク評価が必要 |
| **活性型ビタミンD** | カルシトリオール | カルシウム吸収を促進 | 血清カルシウム値を定期確認 |
| **副甲状腺ホルモン** | テリパラチド | 骨形成を促進する（強力な新薬） | 週2回注射の自己管理が必要 |

**訪問看護師の役割：**
- 毎回の訪問で服薬状況を確認
- 食道炎など副作用の早期発見
- 正しい服用方法の説明・指導
- 血液検査の採取・結果の説明

定期的な血液検査（3～6ヶ月ごと）と骨密度検査（1～2年ごと）により、治療の有効性を確認します。これらについては**主治医の指示に従ってください**。

### カルシウムとビタミンDの栄養管理

骨を丈夫に保つには、カルシウムとビタミンDの十分な摂取が不可欠です。

**推奨摂取量（65歳以上）：**
- カルシウム：1,000～1,200 mg/日
- ビタミンD：600～800 IU/日（または15～20 mcg/日）

**カルシウムが豊富な食品（100gあたりの含有量）：**
- 干しエビ：2,000 mg
- 煮干し：1,500 mg
- 牛乳：110 mg
- ヨーグルト：100 mg
- 小松菜：150 mg
- 豆腐：150 mg
- チーズ：600 mg

**ビタミンDが豊富な食品：**
- サケ（生）：34 mcg/100g
- マグロ（中トロ）：32 mcg/100g
- シイタケ（乾燥）：12.7 mcg/100g
- 卵黄：5.6 mcg/100g
- 牛乳（強化）：1.5～2.5 mcg/200ml

**実践的なポイント：**
- 朝食時に牛乳やヨーグルトを習慣づける
- 週に2～3回は魚を摂取する
- 週3日以上、日中に15～20分の日光浴（ビタミンD合成促進）
- 栄養バランスが心配な場合は、訪問栄養士の相談を利用

嚥下困難や食事摂取が困難な場合は、栄養補助食品（高カルシウム・ビタミンD配合）の活用を**主治医や栄養士に相談**してください。

## 疼痛管理の実践的アプローチ

### 骨折後の疼痛の特徴と段階

骨折後の痛みは、治癒過程に応じて変化します。その段階を理解することで、適切な対応ができます。

**骨折後の疼痛の段階**

1. **急性期（0～4週間）**
   - 特徴：強い痛みと腫脹
   - 原因：骨の破断、周囲組織の炎症
   - 対応：冷却、固定、鎮痛薬による痛み管理

2. **修復期（4週～3ヶ月）**
   - 特徴：徐々に痛みが軽減するが、運動時に増悪
   - 原因：骨癒合過程の進行
   - 対応：段階的な運動開始、温熱療法、非オピオイド鎮痛薬

3. **リモデリング期（3ヶ月～1年以上）**
   - 特徴：残存痛（神経痛様）、動きの制限
   - 原因：神経の圧迫や瘢痕組織形成
   - 対応：理学療法、必要に応じてオピオイド（麻薬性鎮痛薬）

### 在宅での疼痛評価方法

効果的な疼痛管理には、**正確な疼痛評価**が欠かせません。以下の方法を定期的に実施します。

**NRS（数値評価スケール）：最も実用的**
- 「痛みの強さを0～10で評価してください」
- 0=痛みなし、10=考えられる最強の痛み
- 高齢者にも理解しやすい

**VAS（視覚的アナログスケール）：より詳細な評価**
- 10cm横棒の左端を「痛みなし」、右端を「最強の痛み」として、
  患者さんの痛みの程度を示す点を記録
- トレンド（改善傾向）が一目瞭然

**訪問看護師の評価チェックリスト：**
- □ 痛みの部位（骨折部か、その周囲か）
- □ 痛みの質
