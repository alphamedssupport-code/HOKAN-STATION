## イントロダクション

在宅で暮らす高齢者の「最近やせてきた」「足が弱くなった」という悩みを抱える家族は多いでしょう。これらの背景には、**低栄養とサルコペニア（加齢に伴う筋肉量・筋力の低下）** という、見過ごされやすくも深刻な課題があります。日本老年医学会の報告によると、在宅高齢者の約17〜50%が低栄養もしくはその危険性を抱えており、転倒・骨折・入院リスクが大きく増加します。本記事では、訪問看護の現場で実践されている **栄養状態の評価方法から、食事支援、運動療法までの包括的な対策** をご紹介します。現在のご家族の状況を改善するための、具体的で実行可能な方法が見つかります。

## 高齢者が低栄養・サルコペニアに陥る理由

### 身体的・生理的要因

高齢者の低栄養は決して「食べる量が少ないから」だけでは説明できません。**加齢に伴う体の変化** が複合的に関係しています：

- **味覚・嗅覚の低下**：加齢とともに味蕾（みらい）の数が減少し、濃い味付けを好むようになり、塩分過剰摂取につながる
- **咀嚼・嚥下機能の衰退**：歯の喪失、唾液分泌減少、嚥下反射の鈍化により、食べる意欲が低下
- **消化吸収機能の低下**：胃酸分泌量減少、腸の蠕動運動（ぜんどうどうき）低下により、栄養素の吸収効率が30〜40%低下
- **ホルモン変化**：グレリン（食欲ホルモン）分泌低下により、空腹感を感じにくくなる
- **代謝の変化**：基礎代謝が年1%程度低下し、同じ活動量では筋肉が減少しやすくなる

### 心理・社会的要因

- **孤食（こっしょく）**：一人で食事をすることで、喫食量が30%程度低下する傾向
- **抑うつ症状**：食欲不振、調理への意欲喪失につながる
- **経済的困難**：年金生活で、タンパク質豊富な肉・魚・卵などの購入が困難
- **社会的孤立**：外出減少に伴う人間関係の希薄化

### 疾患・薬剤関連要因

- **多剤併用**：複数の薬剤による相互作用で食欲不振、吐き気が生じる
- **消化器疾患**：胃炎、逆流性食道炎、慢性便秘による食事摂取の制限
- **認知症**：食べ方を忘れる、食事を拒否する行動
- **がん悪液質（あくえきしつ）**：がん患者に見られる急速な筋肉・脂肪喪失

## 低栄養・サルコペニアの症状と診断基準

### 早期発見のための症状チェック

以下の症状が複数当てはまる場合は、栄養評価が必要です：

| 症状カテゴリ | 具体的な症状 |
|------------|----------|
| **体重変化** | 6ヶ月で体重が5kg以上減少 / 最近1ヶ月で1kg以上減少 |
| **身体機能** | 階段の上り下りが困難になった / 20m歩くのに30秒以上かかる |
| **筋肉量** | ふくらはぎが細くなった（10cm未満） / 握力が低下 |
| **栄養摂取** | 1日3食摂取できない / 食事量が半分以下に減少 |
| **日常生活** | トイレまで自分で行けなくなった / 入浴が一人でできない |
| **一般症状** | 倦怠感（けんたいかん）が続く / 風邪をひきやすくなった |

### 医学的診断基準

#### **低栄養の判定基準**（日本老年医学会）

| 指標 | 診断基準 |
|-----|---------|
| **BMI（体重kg ÷ 身長m²）** | 18.5未満で低体重 / 高齢者は20未満で注意 |
| **血中アルブミン値** | 3.5g/dL未満で栄養不良 / 3.0g/dL未満で重度 |
| **プレアルブミン値** | 20mg/dL未満で低栄養リスク |
| **総リンパ球数** | 1200/μL未満で免疫栄養不良 |

#### **サルコペニアの診断基準**（EWGSOP2基準）

世界老年医学会は以下の3段階の診断を提唱しています：

1. **疑い段階**：握力低下（男性<27kg、女性<16kg）または歩行速度低下（<0.8m/秒）
2. **サルコペニア確定**：上記に加えて筋肉量低下を確認（DEXA、BIA法など）
3. **重症サルコペニア**：筋肉量＋筋力＋筋機能の全てが低下

## 訪問看護での栄養評価・スクリーニング

### 初期評価での5つの重要項目

訪問看護師が初回訪問時に実施すべき栄養評価を、具体的な手順で説明します。これは医師の指示や栄養士との協働で行われるべき業務です。

#### **1. MNA-SF（簡易栄養評価スケール）の実施**

MNA-SFは、わずか5～10分で栄養状態をスクリーニングできるツールです：

**質問項目と採点例：**

```
Q1. 最近3ヶ月間に、食欲不振や体調不良で食事量が減りましたか？
    ① 食べる量が著しく減少 → 0点
    ② 食べる量がやや減少 → 1点
    ③ 食べる量に変化なし → 2点

Q2. 最近3ヶ月間の体重変化は？
    ① 3kg以上の体重減少 → 0点
    ② 1～3kgの体重減少 → 1点
    ③ 体重変化なし → 2点

Q3. 移動能力は？
    ① 寝たきり または椅子座位のみ → 0点
    ② 移動に介助必要 → 1点
    ③ 自力移動可能 → 2点

Q4. 精神的ストレスまたは急性疾患があった？
    ① はい → 0点
    ② いいえ → 2点

Q5. 神経心理学的問題は？
    ① 重度の認知症または抑うつ → 0点
    ② 軽度の認知症 → 1点
    ③ なし → 2点
```

**判定結果：**
- 12点以上：栄養状態良好
- 8～11点：栄養不良の危険性あり → 詳細評価に進む
- 7点以下：栄養不良と判定 → 栄養士介入・医師報告が必須

#### **2. 食事摂取量の客観的評価**

「どのくらい食べていますか？」という質問だけでは不正確です。**実際の3日間食事記録** を取ることが推奨されます：

**実施方法：**
- 訪問時に前日の全食事内容をヒアリング
- 可能であれば写真記録を依頼
- 外出時の食事、間食、栄養補助食品も含める
- 栄養士に送付し、エネルギー・タンパク質量を計算

**評価ポイント：**

| 項目 | 評価内容 |
|-----|--------|
| **エネルギー摂取** | 高齢者の推奨は25～30kcal/kg/日。体重60kgなら1500～1800kcal/日が目安 |
| **タンパク質摂取** | 推奨1.0～1.2g/kg/日。体重60kgなら60～72g/日（鶏卵約10個相当） |
| **食事の多様性** | 毎日同じものばかり食べていないか |
| **嚥下機能に合わせた形態** | 食事が本人の咀嚼・嚥下能力に合致しているか |

#### **3. 握力・下肢筋力の測定**

**握力測定：**
- 握力計を使用、両手各3回測定し平均値を記録
- 判定基準：男性27kg未満、女性16kg未満で筋力低下
- 簡便法：握力が減少傾向（前回比10%以上低下）でサルコペニアリスク増加

**下肢筋力テスト：**
- **5回座り立ちテスト**（椅子から5回立ち上がるのにかかる時間）
  - 12秒以下：良好
  - 12～15秒：低下の可能性
  - 15秒以上：筋力低下あり → リハビリ強化が必要

#### **4. ふくらはぎ周囲径（下腿周囲）の測定**

膝から10cm下の地点を測定：
- 34cm以上：良好
- 30～33cm：低下の可能性
- 30cm未満：フレイル（虚弱）の可能性が高い

**測定のコツ：**
- 毎回同じ位置で測定（ペンで目印をつけることもある）
- 毎月1回程度測定し、減少傾向を記録
- 非利き脚を測定（浮腫などで変動しやすい利き脚は避ける）

#### **5. 医学的情報の確認**

| 確認項目 | 目的 |
|--------|------|
| **服用薬剤一覧** | 食欲不振や嘔気を引き起こす薬剤の確認 |
| **既往歴** | 嚥下困難、消化器疾患などの有無 |
| **アレルギー・嗜好** | 食事提供時の注意事項 |
| **摂食補助用具** | 義歯の装着状況、スプーンやフォークが使用可能か |

## 訪問看護による食事支援の実践戦略

### 段階的な栄養改善プログラム

低栄養状態の改善は、一度に大きな変化を求めるのではなく、**段階的で小さな成功の積み重ね** が鍵となります。

#### **ステップ1：現状把握と目標設定（初期2週間）**

**訪問看護師の役割：**
- 本人・家族の栄養に関する価値観や課題を傾聴
- 実現可能な目標を共に設定（例：「体重2kg増加を3ヶ月で達成」）
- 主治医・栄養士と情報共有

**このステップでの「小さな成功」の例：**
- 毎日同じ時間に食事する習慣づけ
- 好物の食材を1品追加する
- 食事の環境を改善（音楽をかける、人と一緒に食べる等）

#### **ステップ2：食事環境・形態の工夫（2週～4週）**

**食事環境の改善：**

| 工夫項目 | 具体的対策 | 期待される効果 |
|--------|---------|----------|
| **食事時間** | 毎日同じ時間に設定 | 胃液分泌のリズムを調整、食欲向上 |
| **食事場所** | 可能なら家族と同じテーブルで | 孤食が解消、喫食量30%増加の報告例 |
| **温度** | 温かい汁物を提供 | 嗅覚刺激により食欲増進 |
| **音環境** | 静かすぎない環境（テレビなど） | リラックス効果で嚥下がスムーズに |
| **照度** | 昼間は自然光、食卓は明るく | サーカディアンリズム改善 |

**食事形態の工夫：**

咀嚼・嚥下機能に合わせた工夫は、無理なく栄養摂取を続けるために不可欠です。日本摂食嚥下リハビリテーション学会のガイドラインに基づいた形態分類：
