## イントロダクション

大切な家族が自宅で療養している時、もし何か異変が起きたら……そんな不安は、在宅医療を選択した多くの家族が抱える心配です。実際、厚生労働省のデータによれば、自宅での療養者は年々増加しており、家族は医療職と共に「生活の場」での急変対応を担う重要な役割を果たしています。しかし、具体的な対応手順や判断基準がわからないと、いざという時に冷静な判断ができません。

この記事では、訪問看護の現場で実際に家族が直面する緊急時の対応を、具体的な手順・判断基準・心構えまで、わかりやすく解説します。訪問看護師との連携方法、119番通報の判断ポイント、そして心理的な準備まで。この知識があれば、家族としての自信が生まれ、いざという時に大切な人を守ることができます。

## 緊急時の判断基準：「何が異常か」を見分ける

自宅療養では、「いつもと違う状態」に気づくことが最初の一歩です。しかし何が緊急で、何が様子見でよいのか、その判断は多くの家族にとって難しい課題です。

### 直ちに119番通報すべき「赤信号」症状

以下のいずれかが見られた場合は、**迷わず119番通報してください**。これらは生命に関わる危険が高い状態です。

| 症状カテゴリ | 具体的な症状 | 判断のポイント |
|-----------|---------|-----------|
| **呼吸** | 呼吸がない、または呼吸が極端に浅い、喘鳴が強い | 5秒以上呼吸が見られない、顔色が紫色になっている |
| **意識** | 意識がない、または呼びかけに反応しない | 肩をたたいて大声で呼んでも目を開けない |
| **胸痛** | 胸を圧迫される痛み、突然の激痛 | 顔面蒼白、冷や汗、左腕の痛みを伴う場合 |
| **脳卒中兆候** | 顔の片側が下がっている、片腕が上がらない、言葉が不明瞭 | 3つ全て当てはまる場合は「脳卒中の可能性」 |
| **出血** | 制御できない出血、吐血、下血 | ガーゼで5分以上押さえても止まらない |
| **窒息** | 気道異物による窒息、または窒息の危険 | 息ができず、話せない状態 |
| **重度の外傷** | 大きな転落、頭部外傷、骨折の可能性 | 意識障害がなくても、著しい痛みや腫れ |

**重要：** 判断に迷ったら、**迷わず119番通報してください**。相談程度なら、先に訪問看護ステーションや医師に連絡してもよいですが、上記症状が明らかな場合は通報優先です。

### 訪問看護師・医師への「電話相談」レベルの症状

以下のような症状がある場合は、落ち着いて訪問看護ステーション・医師に電話連絡し、指示を仰ぎましょう。

- 発熱（38℃以上、または本人の平時より1℃以上高い）
- 嘔吐や下痢（数回程度、脱水の兆候がない場合）
- 軽度の意識変容（いつもより反応が鈍い、寝てばかりいる）
- 食事摂取量の著しい低下
- 尿量や便の変化（量が少ない、色が濃いなど）
- 軽度の出血（歯磨き時の出血、鼻血など、自然に止まる程度）
- 軽い転倒後、痛みはあるが意識・呼吸に問題がない

**判断のコツ：** 「いつもと違うけど、生命に関わるほどではない」という状態です。このレベルは、訪問看護師のアドバイスで自宅対応できることも多いです。

## 緊急時対応フロー：その時、家族は何をするか

いざという時の慌てやすさを減らすため、実際の対応手順を具体的なフローで示します。壁に貼り出すか、スマートフォンに保存しておくことをお勧めします。

### 【ステップ1】最初の60秒：状態確認と通報判断

```
状態をすばやく確認
    ↓
119番通報すべき赤信号症状がある？
    ├→ YES：「119番通報」へ進む
    └→ NO：「訪問看護師・医師に連絡」へ進む
```

**実際の確認項目（20秒以内で実施）：**

1. **呼吸確認**
   - 胸や腹が動いているか（5秒見る）
   - 聴診器がなくても、頬を近づければ呼気を感じられます

2. **意識確認**
   - 肩をやさしくたたきながら「〇〇さん！」と大声で呼ぶ
   - 目を開けるか、言葉に応答するか確認

3. **皮膚色確認**
   - 顔色が紫色（チアノーゼ）、蒼白、赤くなっていないか

4. **体の動き確認**
   - 痙攣（けいれん）していないか
   - 片側の手足が動かないか

### 【ステップ2】119番通報する場合の対応

**通報時に冷静さを保つコツ：**

訪問看護師や医療職が現場に多くいるため、落ち着きを失いがちです。しかし、119番通報時は正確な情報が救急車の出動判断を左右します。以下の情報を簡潔に伝えてください。

**119番通報時の「黄金情報」：**

| 項目 | 例 |
|------|---|
| **住所** | 「〇〇県〇〇市△△町1-2-3」（番地まで正確に） |
| **患者情報** | 「80代女性、在宅療養中」 |
| **現在の状態** | 「意識がない」「呼吸がない」「胸が痛いと言って倒れた」 |
| **その他の情報** | 「持病は心臓病です」「現在ペースメーカーを装着しています」 |

**実際の通報例：**
> 「〇〇市△△町1番地に住む80代女性です。さっき突然意識を失いました。呼吸は浅いようです。心臓病の既往があります。お願いします。」

**通報後の対応：**

1. **回線は切らない**
   - 指令室から指示が出ることがあります
   - 心肺蘇生（CPR）の指導を受けることもあります

2. **玄関の施錠を解除しておく**
   - オートロックの場合は、到着時にインターホンで応答

3. **患者さんのそばを離れない**
   - 万が一の対応が必要なため、見守りを続ける

4. **訪問看護師・医師には別途連絡**
   - 通報後、訪問看護師にも連絡してください
   - 医師の指示を先に受けていた場合、その情報が医療チームの対応をスムーズにします

### 【ステップ3】訪問看護師・医師に連絡する場合

**連絡時のポイント：**

- **タイムスタンプを記録する**
  - 「午前10時15分に気づいた」という正確な時刻情報は重要です

- **「いつから」「どのくらい続いているか」を説明する**
  - 「さっき気づいた」「昨夜からずっと」といった時間軸の情報

- **「他に何か変化していないか」を一緒に確認する**
  - 発熱、下痢、嘔吐など関連する症状の有無

**実際の連絡例：**
> 「おはようございます。〇〇です。母が昨夜から38.5℃の熱があり、今朝も続いています。食事はまったく口に入らず、トイレに行く回数も少ないようです。何か対応することはありますか？」

## 在宅医療の連携体制：あらかじめ構築しておくこと

緊急時の対応の成否は、「その時の判断」だけでなく、「事前の準備」で大きく左右されます。実際に訪問看護の現場では、家族が日頃から医療チームとしっかり連携できている家庭ほど、落ち着いた対応ができています。

### 連絡先情報のまとめ方：「緊急連絡票」の作成

自分たちの手元にいつでも参照できるようにしておくべき情報を、一枚の紙にまとめます。

**「緊急連絡票」に記載すべき内容：**

```
═══════════════════════════════════════
         緊急連絡票（患者さんの名前：    ）
═══════════════════════════════════════

【本人の基本情報】
・氏名：
・年齢：
・主な持病：
・アレルギー（薬剤・食物）：
・現在の服用薬：（医師名、処方の主な内容）
・身体障害者手帳・介護保険証の有無：

【医療チームの連絡先】
□ 訪問看護ステーション
　 名称：
　 連絡先：       通常時 / 夜間対応
　 担当看護師：

□ 主治医（医師）
　 氏名：
　 医院・病院名：
　 連絡先：       通常時 / 緊急時

□ 調剤薬局
　 名称：
　 連絡先：

□ ケアマネジャー
　 氏名：
　 事業所名：
　 連絡先：

【主な健康情報】
・普段の体温：     ℃
・普段の血圧：   ～   mmHg
・普段の心拍数：    回/分
・普段の呼吸数：    回/分

【緊急時の判断・対応】
赤信号症状が見られたら：119番通報 → 訪問看護師に連絡
黄色信号症状が見られたら：訪問看護師に連絡 → 指示を仰ぐ

═══════════════════════════════════════
```

このフォーマットを冷蔵庫に磁石で貼り出すか、壁に貼っておくと、急な事態でも誰でも情報にアクセスできます。

### 訪問看護師との「事前打ち合わせ」で確認すべき項目

多くの家族は「何かあったら連絡するだけ」と思っていますが、実は事前にしっかり相談しておくべき項目があります。訪問看護師との信頼関係が、緊急時の対応の質を大きく高めます。

**初回訪問時（または定期相談時）に確認すべき質問：**

1. **「この患者さんにとって、何が『緊急』ですか？」**
   - 例えば、認知症で独特な行動がある場合、その背景を理解しておく
   - 末期がんの患者さんの場合、「苦痛の緩和」が最優先かもしれません

2. **「夜間や休日に問題が生じた時、どこに連絡すればいいですか？」**
   - 訪問看護ステーションが24時間対応か、限定的か
   - 医師の緊急対応の体制はどうなっているか

3. **「自宅でできる応急処置は何ですか？」**
   - 例：発熱時は冷却シートで冷やしてもいいか
   - 例：嘔吐した時の体位変換の方法

4. **「こういう時は119番ですか、連絡ですか？」**
   - 実際のシナリオで確認すると、より実用的です
   - 「もし明け方に急に呼吸がおかしくなったら？」

5. **「家族の役割は何ですか？」**
   - 毎日の記録（体温、食事摂取、排便など）で訪問看護師の判断を支援する
   - その記録の形式（ノート、アプリなど）
