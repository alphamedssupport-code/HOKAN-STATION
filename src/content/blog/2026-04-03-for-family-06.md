## 在宅酸素療法とは｜基本知識

### 在宅酸素療法の目的と対象者

在宅酸素療法（HOT：Home Oxygen Therapy）とは、**酸素を自宅で継続的に吸入する医療行為**です。血液中の酸素濃度が低下している患者さんが対象となります。

**対象となる患者さんの主な疾患：**

- 慢性閉塞性肺疾患（COPD）
- 間質性肺炎
- 肺線維症
- 慢性肺性心
- 肺高血圧症
- 重症な心不全

医学的には、安静時の動脈血酸素分圧が基準以下、もしくは特定の運動負荷時に酸素低下が見られる場合に、医師によって処方されます。日本では、**医療保険が適用される重要な治療**で、適切に使用すれば寿命延長や生活の質向上が期待できます。

### 在宅酸素療法の重要性

呼吸器疾患では、肺機能の低下によって体内の酸素取り込み能力が落ちます。この状態が続くと：

- **心臓への負担が増加**：酸素不足を補うため心拍数が上がり、不整脈のリスクが高まる
- **脳機能の低下**：認知機能や判断力の低下
- **運動耐容能の減少**：日常生活の活動範囲が著しく縮小
- **感染症のリスク増加**：免疫機能の低下

適切な在宅酸素療法により、これらの症状を改善し、患者さんが自宅で充実した生活を送ることができるようになります。

## 在宅酸素療法の主な酸素供給方式

### 酸素濃縮器（酸素コンセントレーター）

**最も一般的な方式**で、空気から酸素を取り出して濃縮し供給します。

**特徴：**
- 電気で動作（電源がないと使用不可）
- ランニングコストが安い
- 騒音がある（通常40-50デシベル）
- 連続的な酸素供給が可能
- 設置スペースが必要（冷蔵庫程度のサイズ）

**家族が注意すべき点：**
- 毎日2時間以上の連続使用を前提に設計されている
- フィルター清掃が定期的に必要（月1-2回の簡単な掃除）
- 周囲に十分な通気性が必要（機器が熱を持つ）

### 液化酸素（液酸）

液体の酸素を保冷して貯蔵し、使用時に気体に変えて供給する方式です。

**特徴：**
- コンパクトで携帯しやすい
- 供給量が多い
- 定期的な補充が必要（医療機関で）
- コストが濃縮器より高い

**家族が注意すべき点：**
- 非常に低温（マイナス183℃）のため、火傷のリスク
- 定期的な補充スケジュール管理が必須
- 移動が多い患者さんに適している

### 圧縮酸素（酸素ボンベ）

高圧ガスを鋼製ボンベに圧縮して貯蔵する方式です。

**特徴：**
- 携帯性に優れている
- 短期使用に向いている
- 使用量に応じて継ぎ足し補充が必要
- 重量がある

**家族が注意すべき点：**
- 落下させないようにしっかり固定
- 高温・炎熱環境を避ける
- 使用量管理のための圧力計確認

## 家族ができる日常的な管理｜チェックリスト

### 毎日チェックすべき項目

在宅酸素療法が安全に機能しているかを確認するために、**家族が毎日行うべきチェック項目**をまとめました。

| チェック項目 | 内容 | 頻度 | 対応方法 |
|:---|:---|:---|:---|
| 機器の電源状態 | 濃縮器のパワーが入っているか、表示ランプが正常か | 毎朝・毎夜 | 異常があれば訪問看護師に連絡 |
| 酸素流量 | 処方された流量に設定されているか | 毎朝 | 患者さんが操作していないか確認 |
| チューブの接続 | 鼻カニューラ（酸素投与チューブ）が正しく接続されているか | 毎日複数回 | 外れているなら接続し直す |
| 鼻カニューラの状態 | ひび割れ、劣化していないか | 毎日 | 破損時は交換（供給企業に連絡） |
| アラーム音 | 異常音がしていないか | 毎日 | 停止しない場合はすぐに連絡 |
| 酸素残量 | ボンベや液酸の残量確認（該当者） | 毎日 | 予定補充日を確認 |
| 周囲環境 | 機器周辺に水分、燃焼物がないか | 毎日 | 安全距離（1m以上）を確保 |
| 患者さんの呼吸状態 | 呼吸困難感、唇の色、意識状態 | 毎日複数回 | 異常があれば医師に報告 |

### 酸素濃縮器のメンテナンス

訪問看護師から教わったメンテナンス方法を実践することで、機器の寿命を延ばし、安全性を確保できます。

**フィルター清掃（月1-2回）：**
1. 機器背面のフィルターカバーを開ける
2. フィルターを静かに取り出す
3. 掃除機で軽くホコリを吸い取る（または水道水で軽く洗う）
4. 完全に乾燥させてから装着
5. 重度の汚れが見られたら、販売企業に交換を依頼

**外観の清掃（週1回）：**
- 軟らかい布で表面と底部のホコリを拭う
- 通気口を塞がないよう注意

**警告：** 機器内部の分解清掃は**絶対に行わないでください**。安全と効能が損なわれます。

### 酸素療法中に避けるべき行動

酸素療法は火災との関連で注意が必要な治療です。**家族全体が安全意識を持つことが重要**です。

**禁止事項：**
- タバコ・喫煙（患者さんの周囲での喫煙は厳禁）
- 裸火の近接使用（ろうそく、線香、ガスコンロの直火）
- オイルヒーターなどの放射熱源
- 酸素吸入中の整髪料・制汗スプレーなどの引火性エアゾール使用
- 静電気を溜めやすい化学繊維の過剰な使用

**推奨事項：**
- 綿素材の衣類を優先的に使う
- 加湿器を使う（静電気低減のため）
- 機器周囲に火災報知器を設置

## 患者さんの身体変化を観察するポイント

### 呼吸状態の評価

家族が毎日観察することで、患者さんの状態悪化を早期発見できます。

**チェック項目：**

**呼吸数：** 安静時の呼吸数が20回/分以上は要注意。通常は12-20回/分です。

**呼吸パターン：** 
- 均等で落ち着いた呼吸が理想的
- 浅くて速い呼吸、または深い呼吸が続く場合は異常のサイン

**チアノーゼ（爪や唇の青紫色）：** 適切な酸素療法が行われていれば見られません。これが出現すれば、酸素流量不足か機器故障の可能性があります。

**労作時呼吸困難：** 階段昇降や歩行時に著しく呼吸が苦しくなることが増えたら、医師に報告が必要です。

### 生活機能の観察

酸素療法の効果判定には、患者さんの日常生活活動（ADL）の変化も重要です。

**チェックリスト：**
- [ ] 階段の上り下りが楽になったか
- [ ] 以前より長く歩けるようになったか
- [ ] 疲労感が軽減したか
- [ ] 認知機能（会話、判断）に改善が見られるか
- [ ] 夜間の睡眠の質が改善したか

改善が見られない場合や悪化している場合は、**酸素流量の調整が必要な可能性**があります。訪問看護師や医師に相談してください。

### 異常の早期発見

以下の症状が見られたら、**早期に医療機関に連絡する必要があります**：

| 症状 | 対応 |
|:---|:---|
| 持続する強い頭痛 | 医師に電話相談 |
| 意識障害（反応が鈍い、意識がない） | 119番救急車 |
| 急激な胸痛 | 119番救急車 |
| 唇や爪が明らかに紫色になった | すぐに訪問看護師に連絡 |
| けいれん | 119番救急車 |
| 吐血・血痰 | 医師に電話相談 |

## 緊急事態への対応方法

### 機器の故障・停止時

在宅酸素療法患者さんの機器故障は、医療的に深刻な状況になりうるため、対応の優先順位が重要です。

**ステップ1：機器トラブルシューティング（1-2分以内）**

| 症状 | 確認項目 | 対応 |
|:---|:---|:---|
| 機器が動作しない | 電源プラグが差さっているか | 差し直す、別のコンセント試す |
| 表示ランプが消えている | ブレーカーが落ちていないか | ブレーカーをリセット |
| アラーム音が鳴っている | 酸素残量は十分か | 液酸・ボンベの残量を確認 |
| 酸素が出ていない | チューブが折れていないか | チューブを伸ばす、つなぎ直す |

**ステップ2：医療機関への連絡（確認後すぐ）**

症状が改善しない、または患者さんの呼吸が苦しくなった場合は：

1. **訪問看護師に電話**（24時間対応の番号がある場合）
2. **酸素供給企業に電話**（機器メーカーのサービス番号）
3. **かかりつけ医に電話**
4. **救急車が必要と判断したら119番**

**ステップ3：携帯酸素ボンベの確保**

- 酸素濃縮器が故障した場合に備え、常に**予備の携帯酸素ボンベを用意**しておくことが望ましい
- 訪問看護師にボンベの使用方法を事前に確認しておく
