## 経管栄養の在宅管理へようこそ

ご家族が経管栄養（けいかんえいよう）での栄養補給が必要になると、「自宅で安全に管理できるだろうか」という不安は当然です。病院での慣れた環境から退院し、初めて自宅での栄養管理に向き合うご家族の戸惑いのお声を、訪問看護の現場でよくお聞きします。

しかし、適切な知識と技術を身につけ、訪問看護師のサポートを受けながら進めていけば、経管栄養は自宅でも安全に管理できるケアです。本記事では、訪問看護師が実際に家族指導で伝える、経管栄養の正しい方法、日常的な注意点、トラブル対応まで、家族だからこそ知っておくべき実践的なポイントをまとめました。

この記事を読むことで、経管栄養管理への不安を減らし、ご家族がご自宅で安心して継続的なケアを提供できるようになります。

## 経管栄養とは｜基礎知識から始める

### 経管栄養の定義と種類

経管栄養とは、口から食事が摂取できない、または摂取が困難な患者さんに対し、鼻や腹部を通して直接胃や小腸に栄養剤を注入する栄養管理方法です。

**主な経管栄養の種類は以下の通りです：**

| 種類 | 挿入方法 | 使用期間 | 特徴 |
|------|--------|--------|------|
| **経鼻胃管（NG管）** | 鼻から食道を通して胃へ | 短期（数週間～3ヶ月） | 挿入・交換が比較的簡単。外見への負担がある |
| **経皮内視鏡的胃瘻（PEG）** | 腹部の皮膚から直接胃へ | 中～長期（数ヶ月～数年） | 固定性が高く、長期管理に適している |
| **腸瘻** | 腹部から小腸（空腸）へ | 中～長期 | 胃機能の低下がある場合に選択される |

ご家族のご家族が現在使用している経管栄養の種類と、それぞれの特性（定期交換時期、管理方法など）については、必ず主治医や訪問看護師に確認してください。

### 自宅での経管栄養管理が必要な主な疾患

経管栄養の在宅管理が必要になる主な疾患には以下のものが挙げられます：

- **脳卒中後遺症**：嚥下機能（えんげきのう＝飲み込む機能）の障害
- **神経難病**（ALS・パーキンソン病など）：進行に伴う嚥下困難
- **認知症**：食事拒否や誤嚥のリスク
- **頭頸部がん術後**：解剖学的な食道変化
- **高齢虚弱**：全身状態の低下に伴う経口摂取困難

「この疾患に経管栄養が適切なのか」「他の選択肢はないか」については、主治医と十分に相談してください。

## 経管栄養剤の選択と準備｜栄養管理の第一歩

### 栄養剤の種類と特性

経管栄養剤は、患者さんの栄養状態、疾患、消化機能に応じて複数の種類から選択されます。正しい栄養剤の選択と準備は、栄養管理の成功を左右する重要なプロセスです。

**主な栄養剤の分類：**

**1. 成分栄養剤（アミノ酸が主成分）**
- 特徴：消化吸収が最も良好。低い浸透圧で胃腸への負担が少ない
- 対象：急性期、消化機能が低下している方
- 例：メディエフ、ピケーアなど
- **成分栄養剤使用時の注意**：開封後は常温で6時間以内、冷蔵で24時間以内に使用

**2. 半消化態栄養剤（中分子栄養）**
- 特徴：バランスの取れた栄養バランス。やや消化が必要
- 対象：安定期の長期管理患者
- 例：エンシュア・エンテリール・グリーンなど

**3. 濃厚流動食（通常の食材を撹拌したもの）**
- 特徴：自然な食事感覚。手作り栄養剤も含む
- 対象：家族の希望がある場合、経腸栄養専用調理法を学んだ家族

**栄養剤の購入・保管・使用のポイント：**

| 項目 | 指示・留意点 |
|-----|----------|
| **処方確認** | 毎月の定期受診時に栄養剤の種類・量・投与方法を確認 |
| **納期・保管** | 医療機関の配送スケジュールを把握。冷暗所で保管（温度15-25℃推奨） |
| **賞味期限** | 毎回、使用前に缶・パックに表示された期限を確認 |
| **温度管理** | 注入直前に温温湯（40℃程度）で温めると注入がスムーズ。熱湯は使用厳禁 |
| **開封後の保管** | 缶製品は開封後8時間以内、パウチ製品は6時間以内に使用 |

### 栄養剤準備の具体的手順

自宅での栄養剤準備は衛生管理が非常に重要です。以下の手順を毎回実践してください。

**準備に必要な物品（チェックリスト）：**

- [ ] 栄養剤（指定されたもの、期限確認済み）
- [ ] シリンジ（注射器型の注入器、通常50mL）
- [ ] 専用の注入セット（チューブ、スタンド）
- [ ] 温かいお湯（40℃程度）
- [ ] 注入ポンプ（医師から指示がある場合）
- [ ] 清潔なタオル・ペーパータオル
- [ ] 消毒液（70%アルコール）
- [ ] 滅菌手袋（不潔操作の場合）

**具体的な準備手順（5ステップ）：**

**【ステップ1】環境整備と手指衛生**
1. 準備する場所は台所またはリビングの清潔な平らな場所
2. 手をぬるま湯と石けんで30秒以上、指の間も含めてよく洗う
3. 清潔なタオルで水気を拭き取る
4. 可能であればアルコール手指消毒液で消毒（乾いた手に15秒程度）

**【ステップ2】栄養剤と物品の確認**
1. 栄養剤の種類が処方箋と一致しているか確認
2. 缶またはパウチの外観に傷や膨張がないか確認
3. 賞味期限が本日を含めて残っているか確認
4. 必要な注入用具がすべて揃っているか確認

**【ステップ3】栄養剤を使いやすい状態に**
1. 缶製品の場合：缶の上部をきれいに拭き、栓を開ける
2. パウチ製品の場合：パウチの開封口を清潔なはさみで切る
3. シリンジまたは注入ポンプ用の容器に必要量を移す
4. 温かいお湯に浸して、栄養剤を人肌程度（35-40℃）に温める

**【ステップ4】投与セットの準備**
1. 注入セットの空気抜き（プライミング）：栄養剤をチューブ内に満たし、気泡を完全に除去
2. 先端をクリップで止めて、準備完了を待つ
3. 患者さんの胃瘻または経鼻胃管の先端が皮膚に固定されているか確認
4. 挿入部周囲の皮膚に異常（発赤・浸出液）がないか確認

**【ステップ5】投与直前の最終確認**
1. 患者さんの意識レベル、体位を確認（半座位または座位が原則）
2. 注入セットと胃瘻の接続部がしっかり固定されているか確認
3. 栄養剤の温度を確認（冷たすぎないか）
4. 周囲に危険物がないか確認

## 栄養剤の注入方法｜安全で正確な技術を習得する

### 注入方法の3つのパターン

経管栄養の注入方法は、患者さんの状態と医師の指示に応じて3つの方法が使い分けられます。ご家族が実施する方法については、必ず訪問看護師から直接指導を受け、実演練習で確認してください。

**【パターン1】シリンジ法（最もシンプル）**

シリンジ法は、50mLまたは100mLの大型シリンジ（注射器）を使用して、重力の力で栄養剤を流す方法です。家庭での実施が最も簡単で、訪問看護が定期的に行うことも多い方法です。

**シリンジ法の手順：**

1. **患者さんの体位を整える**
   - 半座位または座位に変更（30度以上の上半身挙上）
   - 注入中も同じ体位を保つ
   - 体位変更が困難な場合は、担当医に相談

2. **シリンジの準備**
   - 50mLシリンジに栄養剤を吸い上げる
   - シリンジ先端を胃瘻または経鼻胃管のチューブに接続
   - 接続がしっかりしているか目視で確認

3. **栄養剤を注入する**
   - シリンジの活塞（かっせん＝スライダー部分）をゆっくり下げる
   - **速度の目安**：1回500mL程度の場合、15～20分かけてゆっくり注入
   - 栄養剤が速すぎると、腹部不快感や下痢の原因になるため注意
   - 患者さんの表情・反応を観察しながら進める

4. **注入後の処理**
   - 栄養剤が完全に流入したら、シリンジを取り外す
   - 胃瘻または経鼻胃管の先端をクリップで止める
   - シリンジを温かい流水で洗浄し、乾燥させる

**シリンジ法が適した対象：**
- 1回の投与量が500mL程度までの患者
- 胃機能がある程度保たれている患者
- 家族による実施が可能な環境にある患者

**【パターン2】重力を利用した注入（スタンド法）**

スタンド法は、注入セット用のスタンドに栄養剤のパウチまたは容器を吊り下げて、重力で栄養剤を流す方法です。連続注入や夜間注入が必要な患者さんに選択されることが多いです。

**重力法の手順：**

1. **注入スタンドの準備**
   - スタンドの高さを調整（栄養剤の容器が患者さんの胸部から30-50cm高い位置）
   - 栄養剤が流れすぎないように調整可能なクリップを用意

2. **注入セットをセット**
   - 栄養剤の容器を専用の注入セット（チューブ付き）に接続
   - チューブ内の空気を完全に抜く（プライミング）
   - クリップでチューブを閉じた状態で準備

3. **患者さんへの接続**
   - 患者さんを半座位または座位に体位変更
   - チューブの先端を胃瘻または経鼻胃管に接続
   - 接続部が外れないように固定テープで補強

4. **注入開始**
   - クリップをゆっくり緩めて栄養剤を流す
   - **流速の目安**：30滴/分～1分間に栄養剤が約100-150mL流れる程度（医師指示に準ずる）
   - 15～30分かけてゆっくり注入するのが標準

5. **注入終了後**
