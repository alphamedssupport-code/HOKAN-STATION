## イントロダクション

親や配偶者が在宅療養を始めると、思いのほか多くの費用がかかることに驚く家族は少なくありません。「訪問看護にはいくらかかるのか」「医療保険でカバーされない費用は何か」「少しでも負担を減らす方法はないか」——こうした不安は、在宅療養を継続するうえで大きなストレスになります。本記事では、在宅療養中にかかる医療費の具体的な内訳、平均的な費用目安、そして実践的な節約方法を詳しく解説します。医療費控除や助成制度も含め、家族が知っておくべき経済的対策をまとめました。適切な情報を得ることで、安心して在宅療養を継続できる環境を整えることができます。

## 在宅療養で発生する医療費の全体像

### 訪問看護費用の仕組み

訪問看護は、自宅に看護師が訪問して医療ケアを行うサービスです。費用は大きく「保険診療分」と「自己負担分」に分かれます。

**保険診療の仕組み：**
訪問看護を利用する際、患者は健康保険の自己負担割合（通常3割）を支払います。残り7割は保険が負担します。ただし月間に利用できる訪問回数や単価には上限が設定されており、それを超えた場合は全額自己負担となります。

**訪問看護の基本料金の目安（1回あたり、令和6年度診療報酬）：**
- 初回訪問：約6,500～7,500円（自己負担：約2,000～2,300円）
- 通常訪問（30分程度）：約3,000～3,500円（自己負担：約900～1,050円）
- 通常訪問（60分程度）：約5,500～6,500円（自己負担：約1,650～1,950円）
- 24時間対応加算：月額約2,000～3,000円の追加

※料金は訪問看護ステーションや地域により異なります。詳細は利用予定のステーションに直接確認してください。

### 月間の訪問看護費用の例

実際の月間費用は、週の訪問回数によって大きく異なります。以下に典型的なケースを示します。

**ケース1：週2回の訪問（月8回）**
- 自己負担額（3割）：約8,000～10,000円/月
- 保険負担分：約18,000～23,000円/月

**ケース2：週3回の訪問（月12回）**
- 自己負担額（3割）：約12,000～15,000円/月
- 保険負担分：約27,000～35,000円/月

**ケース3：週5回以上の訪問（月20回以上）**
- 月間上限額に達することが多く、上限を超えた分は全額自己負担
- 上限額の目安：約50,000～60,000円/月（自己負担のみ）

### その他の訪問医療サービスの費用

訪問看護以外にも、在宅療養には多くのサービス費用が発生します。

**訪問診療（医師による往診）：**
- 月2回程度：約3,000～5,000円/月（自己負担）
- 緊急往診時：別途加算

**訪問リハビリテーション：**
- 1回あたり：約600～1,500円（自己負担）
- 週1～2回が一般的

**在宅薬剤指導・調剤：**
- 調剤基本料：約200～400円/月（自己負担）
- 在宅薬剤管理指導：約200～300円（自己負担）
- 薬代（別途、保険適用後の自己負担）

## 在宅療養の月間費用の平均的な内訳

### 患者の状態別・月間費用の目安

在宅療養にかかる総費用は、患者の要介護度や医療的ニーズによって大きく異なります。以下は典型的なケースです。

**軽度ケース（要介護1～2、週2回訪問看護）：**
| 項目 | 月間自己負担額 |
|------|-----------------|
| 訪問看護（3割） | 8,000～10,000円 |
| 訪問診療 | 3,000～5,000円 |
| 薬代（自己負担分） | 2,000～4,000円 |
| **合計** | **13,000～19,000円** |

※介護保険サービス（介護士による身体介護など）は別途

**中度ケース（要介護3～4、週3～4回訪問看護）：**
| 項目 | 月間自己負担額 |
|------|-----------------|
| 訪問看護（3割） | 12,000～18,000円 |
| 訪問診療 | 5,000～8,000円 |
| 薬代（自己負担分） | 3,000～6,000円 |
| 訪問リハビリ（週1回） | 2,000～5,000円 |
| **合計** | **22,000～37,000円** |

**重度ケース（要介護5、24時間対応、週5日以上訪問看護）：**
| 項目 | 月間自己負担額 |
|------|-----------------|
| 訪問看護（上限超過分含む） | 30,000～60,000円 |
| 訪問診療（頻回） | 10,000～15,000円 |
| 薬代（多剤） | 5,000～10,000円 |
| 医療処置用品 | 5,000～15,000円 |
| **合計** | **50,000～100,000円** |

### 医療保険の自己負担限度額（高額療養費制度）

月間の医療費が高額になった場合、「高額療養費制度」により自己負担額が軽減されます。

**70歳未満の場合（標準的な年収層）：**
- 月間医療費が約88,000円を超えた場合、超過分の一部が払い戻される
- 月間上限額の目安：約87,000円

**70歳以上の場合：**
- 一般的な所得層：月間上限額は約18,000円
- 高所得者：月間上限額は約252,000円程度
- 低所得者：月間上限額は約8,000円

**高額療養費の計算例（70歳未満、年収370万～770万の場合）：**
医療費（保険診療分）の合計が400,000円だった場合
- 自己負担額（3割）：120,000円
- 高額療養費で返金される額：120,000円 - 87,430円 = **32,570円**
- 実際の負担額：**87,430円**

高額療養費は申請により返金されます。詳細は加入している健康保険組合に確認してください。

## 介護保険サービスの費用と利用方法

### 介護保険の自己負担の仕組み

在宅療養では、医療保険だけでなく介護保険を組み合わせて利用することが多いです。介護保険サービスは、患者の要介護度によって月間の利用限度額（支給限度額）が決まります。

**要介護度別・支給限度額（月間、令和6年度）：**
| 要介護度 | 支給限度額 | 自己負担1割の場合 |
|---------|-----------|-------------------|
| 要介護1 | 167,650円 | 16,765円 |
| 要介護2 | 197,050円 | 19,705円 |
| 要介護3 | 270,480円 | 27,048円 |
| 要介護4 | 309,380円 | 30,938円 |
| 要介護5 | 362,170円 | 36,217円 |

※支給限度額を超えた部分は全額自己負担となります

### 介護保険で利用できる主なサービスと費用

**訪問介護（ホームヘルパーによる身体介護・生活援助）：**
- 30分未満：約300～500円（1回）
- 30分以上1時間未満：約500～700円（1回）
- 1時間以上1時間半未満：約700～900円（1回）

**訪問入浴介護：**
- 1回あたり：約1,000～1,500円（自己負担1割の場合）

**通所介護（デイサービス）：**
- 1日あたり：約1,000～2,000円（自己負担1割の場合、送迎別途）

**福祉用具レンタル（介護ベッド・車椅子など）：**
- 月額：約500～3,000円（自己負担1割の場合）

### 介護保険と医療保険の組み合わせ利用

同じサービスでも、医療保険と介護保険のどちらが適用されるかで費用が変わります。

**訪問看護の場合：**
- 医療的なケア（カテーテル管理、インスリン注射など）→ 医療保険が優先
- 身体介護のみ（排泄介助、入浴介助など）→ 介護保険での利用も可能
- どちらで利用するかは、患者の状態と家族の状況に応じて相談で決定します

## 保険でカバーされない費用と自由診療

### 医療保険の給付対象外となる費用

在宅療養では、医療保険でカバーされない費用も多く発生します。

**主な対象外費用：**

**①差額ベッド代に相当する費用**
- 訪問看護の利用時間超過分（支給限度額超過後）：全額自己負担
- 24時間対応加算を選択した場合の上乗せ費用の一部

**②医療用品・用具の購入費（保険適用外のもの）**
- 高性能な尿便器やポータブルトイレの購入（レンタルではなく購入の場合）
- 医療用の特殊なマットレスやクッション：5,000～30,000円
- 吸収性紙おむつ（医療保険では支給されない）：月額3,000～10,000円

**③自費の訪問介護・生活支援**
- 支給限度額を超えた分の訪問介護：全額自己負担
- 介護保険対象外の家事援助（庭の手入れなど）

**④自由診療・自費治療**
- 保険外の診療行為：全額自己負担
- 漢方医学など自由診療：5,000～20,000円/月

**⑤その他の実費**
- 医療機関への通院送迎費
- 食事療法用の特別食
- アロマテラピーなどの補完代替療法

### 在宅療養に必要な費用の総額シミュレーション

保険でカバーされない費用を含めた、実際の月間総額をシミュレーションしましょう。

**シミュレーション：脳梗塞後遺症の患者（要介護3、週3回訪問看護）の場合**

| 項目 | 金額 |
|------|------|
| **医療保険分** |  |
| 訪問看護（3割） | 12,000円 |
| 訪問診療 | 5,000円 |
| 薬代 | 4,000円 |
| **介護保険分** |  |
| 訪問介護（月15時間） | 7,500円 |

このうち医療保険・介護保険で支える額は約30,000円程度で、残り10,000～11,000円は家族の実費負担です。

## 医療費を節約するための具体的な方法

### 医療費控除の活用

医療費控除は、家族の経済的負担を軽減する重要な制度です。

**医療費控除の基本：**
- 1年間（1月～12月）に支払った医療費が100,000円を超えた場合、確定申告により所得税の控除を受けられます
- 控除対象額 = （医療費の合計 - 100,000円） × 所得税率
- 最大で約200万円まで控除可能

**医療費控除に含められる費用（訪問看護・在宅医療関連）：**
- ✓ 訪問看護の自己負担分
