## イントロダクション

親が重い病気と診断された時、「子どもにどう説明しよう…」という悩みを持つご家族は少なくありません。小児がんや長期療養が必要な病気の親を持つ子どもたちは、親の様子の変化を敏感に感じ取り、隠されている事実について不安や想像を膨らませてしまうことがあります。医学的な説明不足よりも、「親が話してくれない」ことが、子どもに大きなストレスや心理的負担をもたらす場合もあります。

本記事では、訪問看護の現場で実践されている、発達段階に応じた病気の説明方法と、家族全体が前向きに関わるためのコミュニケーション戦略をお伝えします。年齢別の具体的な言葉がけ、準備のチェックリスト、そして親自身のメンタルケアまで、実践的な情報を網羅しました。

## 1. なぜ子どもに「親の病気」を説明することが大切なのか

### 1-1. 子どもの心理発達と情報隠蔽のリスク

親が重病と診断されても、その事実を子どもに隠そうとするご家族もいます。しかし医学心理学の研究によれば、**子どもは親の様子から何かが起きていることを察知し、隠されている情報について独自に「想像」を始めます**。この想像は実際の事実よりも、子どもの不安心理によって色付けされるため、本当のことより悪く考えてしまう傾向があります。

日本小児がん研究グループの調査では、親の病状について「正確で年相応の説明を受けた子ども」と「説明を受けなかった子ども」を比較した場合、前者の方が以下の点で心理的安定性が高かったと報告されています：

- **不安感の低減**：毎日の行動が予測可能になり、得体の知れない恐怖が減少
- **信頼関係の維持**：親が「話してくれた」という体験が、絆を深める
- **行動上の問題低減**：説明不足グループに比べ、学校不適応や行動障害の発症率が約3割低い

### 1-2. 訪問看護の現場から見た「説明の効果」

訪問看護師たちは日々、家族と関わる中で、子どもへの適切な説明がもたらす変化を目の当たりにしています。以下は、実際に説明後に見られる子どもの変化です：

- **親への協力的態度**：「母さんを手伝いたい」という主体的な行動が増加
- **学校や友人関係の安定**：親のことで頭がいっぱいになるのではなく、友人との関係に集中できる
- **質問の増加**：逆に知りたいことを積極的に聞くようになり、親子間の対話が増える

訪問看護師は、まさにこの「説明のプロセス」をサポートする重要な役割を担っています。

### 1-3. 発達段階による情報処理能力の違い

子どもが「大人と同じように情報を理解できる」と考えるのは、親の落とし穴です。発達心理学の標準的な知見によれば、年齢によって大きく異なります：

| 年代 | 病気の理解度 | 主な特徴 |
|------|------------|--------|
| **3～5歳（幼児期）** | 魔法的思考段階 | 病気を道徳的な罰と考えたり、自分が原因だと思うことがある |
| **6～8歳（小学低学年）** | 具体的理解段階 | 医学的説明をある程度理解できるが、細節より「簡潔性」を重視 |
| **9～12歳（小学高学年）** | 論理的理解段階 | より詳細で科学的な説明に対応でき、問題解決への参加意欲が高い |
| **13歳以上（思春期）** | 抽象的思考段階 | 長期的な予後や人生への影響を理解でき、自分の役割や将来について考える |

## 2. 年代別：子どもへの病気説明の具体的な言葉がけ

### 2-1. 3～5歳（幼児期）の子どもへの説明

**この時期の子どもの特徴：**
- 言葉は話せるが、時間や因果関係の概念が発達途上
- 親の不安を「感じ取る」が、詳しい理由は理解しづらい
- 自分が「何か悪いことをしたから」親が病気になったと考える傾向（魔法的思考）

**おすすめの説明方法：**

説明は以下の3ステップで行います：

**ステップ1：シンプルな事実を伝える**
```
「お母さんの体の中に、悪い細胞が増えてきちゃったんだ。
医者さんが、その悪い細胞をやっつける薬をくれて、
今、お母さんはその薬を飲んでいるんだよ。」
```

このとき、「がん」という言葉を避ける必要はありませんが、難しすぎない言葉選びが重要です。

**ステップ2：子どもの不安を払拭する**
```
「これはね、〇〇(子どもの名前)が何か悪いことをしたから
なったわけじゃないんだよ。
お母さんの体の細胞が、自分たちの気持ちに気付かなくなっちゃったんだ。」
```

この説明により、子どもが自分を責める魔法的思考を緩和します。

**ステップ3：日常への組み込み**
```
「だから、これからもお母さんはお医者さんに通うよ。
時々、いつもと違う様子かもしれないけど、
これはお母さんが頑張ってくれているしるしなんだよ。」
```

**避けるべき表現：**
- ❌ 「お母さんはね、もう死ぬかもしれないんだ」
- ❌ 「これ以上話すと、君が病気になるかもだから秘密だよ」
- ❌ 医学用語を多用した詳細な説明

### 2-2. 6～8歳（小学低学年）の子どもへの説明

**この時期の子どもの特徴：**
- 具体的で論理的な思考が発達し始める
- 「なぜ？」という質問が増え、因果関係に興味を持つ
- 友人との関係が親以上に重要になり始める時期

**おすすめの説明方法：**

この年代には、少し科学的な説明を混ぜると理解と納得が深まります。

**説明テンプレート：**
```
「お父さんの体には『細胞』という小さなつぶがいっぱいあるんだ。
通常は、その細胞たちが助け合って、体を健康に保ってくれる。
でも時々、その細胞の中に『ルールを守らない細胞』が出てくることがあるんだ。
それががんなんだよ。

今、お父さんは医者さんの助けを借りて、
その『ルールを守らない細胞』をやっつけるための
『抗がん剤』という薬を飲んでいるんだ。
この薬は、強い力を持ってるから、副作用という
『お父さんが疲れたり、吐き気がしたりする』ことも起きるんだ。

でも、これはお父さんが頑張ってくれている証拠なんだよ。」
```

**友人への説明の練習：**
この時期の子どもは、学校で友人から「お母さんなぜ来ないの？」と聞かれる可能性があります。事前に「もし友人に聞かれたら、こう言ってもいいよ」と練習しておくことが重要です。

```
親のセリフ：「もし友達に聞かれたら『お母さんは病気の治療をしてるんだ』
『でも頑張ってくれてるから、僕も応援してるんだ』って言ってもいいんだよ。
それ以上話したくなかったら『それ以上は話したくない』ってでいいんだ。』」
```

### 2-3. 9～12歳（小学高学年）の子どもへの説明

**この時期の子どもの特徴：**
- より抽象的で論理的な思考が可能
- 親の役割の変化を理解し、家族の一員として「何かできることはないか」と考え始める
- 同年代との比較が強まり、「自分たちの家族は特別（違う）」という認識が生まれ始める

**おすすめの説明方法：**

この年代には、医学的な正確さと、感情的な支援の両方が必要です。説明会や家族療法のセッションで、訪問看護師や医療心理士と一緒に説明をすることが効果的です。

**説明のポイント：**

1. **医学的な正確さを保つ**
```
「お父さんは『大腸がん』という病気を持っています。
これは、大腸という腸の一部に悪い細胞が増えた状態です。
今、抗がん剤という薬と、放射線治療という方法で、
その悪い細胞をやっつけようとしています。」
```

2. **予後の説明（医師の指示に基づき）**
```
「医者さんの見立てでは、この治療により、
今後5年間は再発の可能性が低いと言われています。
ただし、医学には100%はないので、
継続的な検査が必要です。」
```

3. **子どもができる役割を示す**
```
「お父さんは治療期間は体がしんどいかもしれません。
でも、君が『今日あった楽しかったことを話してくれる』とか、
『一緒に映画を見る』とか、そういうことが、
お父さんの心を元気にしてくれるんだ。」
```

**重要な注意：** 予後（治療の見通し）や詳細な医学情報については、必ず主治医と事前に相談し、「どこまでどう説明するか」を決めてから子どもに伝えてください。

### 2-4. 13歳以上（思春期）の子どもへの説明

**この時期の子どもの特徴：**
- 親の死亡の可能性を理解する能力が発達
- 自分の人生設計や進学などについて考え始め、親の病気がそれに影響する可能性を認識
- 感情的な距離を取ろうとしながらも、深い関係を求める矛盾した欲求がある

**おすすめの説明方法：**

思春期の子どもには、大人並みの情報提供と尊重が必要です。ただし、すべてを任せるのではなく、「親が決めた範囲内で」情報を共有する形をとります。

**説明のフレームワーク：**

1. **個別面談を設定する**
親子だけでなく、訪問看護師やソーシャルワーカーなど、中立的な第三者が同席することが理想的です。

2. **子どもの希望を確認する**
```
「これからお父さんの病気について、いろいろな話をしたいと思ってるんだ。
でも、君がどこまで知りたいのか、何が聞きたいのか、
まずそれを聞かせてもらいたいんだ。」
```

3. **医学的情報を正確に伝える**
- 診断名と病期（ステージ）
- 治療の方法と期間
- 予想される副作用
- 医師が想定している予後（可能なら医師から本人に直接）

4. **感情的なサポートを同時に行う**
```
「今、いろいろな気持ちがあるかもしれないね。
怖いと感じるのも、怒りを感じるのも、全部正常な反応だんだ。
どんな気持ちでも、僕たちはそれを受け入れるよ。」
```

5. **今後の家族の在り方について話し合う**
- 親が健康な時期にやっておきたいこと
- 治療期間中に子どもに期待できることと、できないこと
- 最悪のシナリオについて（話題が重ければ、段階的に）
