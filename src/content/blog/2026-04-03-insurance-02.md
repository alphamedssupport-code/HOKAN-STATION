## イントロダクション

「親が介護が必要な状態になったけれど、何から始めたらいいのか分からない」「介護認定ってどうやって申請するの？」——そんな不安や疑問を抱える方は少なくありません。介護認定（要介護認定）は、介護保険サービスを利用するための最初の重要なステップです。訪問看護など自宅での介護サービスを受けるにも、この認定が必須となります。本記事では、介護認定の申請方法から要介護度の基準、実際の手続きの流れまで、現場の訪問看護師目線での実践的な情報を、初心者にもわかりやすく解説します。

## 介護認定とは｜基本を理解する

### 介護認定制度の目的と意味

介護認定（正式には「要介護認定」「要支援認定」）とは、被保険者が介護保険サービスを利用する際に、その人の心身の状態がどれくらいの介護を必要とするかを公式に判定する制度です。2000年に介護保険制度が開始された際に創設され、現在では全国約800万人が何らかの認定を受けています。

この認定により、**介護費用の大部分が介護保険でカバーされるため**、個人負担を減らしながら必要な介護サービスを受けることができます。認定がなければ、訪問看護や訪問介護などのサービスは全額自己負担となるため、経済的負担が大きく異なります。

### 介護保険の加入対象者

介護保険の加入対象は以下の通りです：

- **第1号被保険者**：65歳以上の全ての人
- **第2号被保険者**：40～64歳で医療保険に加入している人（特定疾病に限定）

**第2号被保険者が認定対象となる特定疾病**には、脳卒中、末期がん、パーキンソン病、関節リウマチ、認知症など16種類が定められています。詳細については、お住まいの市区町村の介護保険課またはお近くの訪問看護ステーションに相談してください。

## 要介護度の基準｜7段階の目安

### 認定の種類と段階

介護認定の結果は、以下の9つの段階に分かれます：

| 認定区分 | 対象者の状態の目安 | 主なサービス対象 |
|---------|---------------------|-----------------|
| **非該当** | 介護を必要としない（自立した状態） | 介護保険サービス対象外。予防事業のみ |
| **要支援1** | わずかな支援で日常生活が送れる | 予防サービス、訪問看護 |
| **要支援2** | 軽い支援を定期的に必要とする | 予防サービス、訪問看護、訪問介護 |
| **要介護1** | 日常生活で部分的な介助が必要 | 訪問看護、訪問介護、デイサービスなど |
| **要介護2** | 日常生活で中程度の介助が必要 | 上記に加え、居宅介護、ショートステイなど |
| **要介護3** | 日常生活で多くの介助が必要 | 施設サービスも含む全サービス対象 |
| **要介護4** | 日常生活で大部分の介助が必要 | 同上 |
| **要介護5** | 日常生活の全てで介助が必要 | 同上 |

### 要介護度を決める判定基準

要介護度は、以下の項目で総合的に評価されます：

**基本調査項目（74項目）：**
- **身体機能**：起立、歩行、階段昇降、転倒のリスク
- **生活機能**：食事、排泄、入浴、着替え、身のこなし
- **認知機能**：認識、意思疎通、記憶、日時の認識
- **行動障害**：問題行動の有無と頻度
- **社会適応**：他者との関係、外出の頻度、役割行動

**特記事項：**
調査員が主観的に「特に注意すべき点」をまとめます。特記事項は判定に重要な役割を果たすため、正確な情報記載が大切です。

**医師意見書：**
かかりつけ医師が記入し、医学的状態の詳細が加味されます。認知症や末期がんなど、調査だけでは把握できない状況が反映されます。

## 介護認定の申請方法｜5ステップで完解説

### ステップ1：申請前の準備

**必要な書類を確認する：**

1. **申請書（要介護認定申請書）**
   - 市区町村の介護保険課窓口またはオンラインで入手可能
   - 本人または家族、成年後見人が記入

2. **介護保険証**
   - 65歳以上の場合、市区町村から送付されています
   - 第2号被保険者は加入している医療保険証の提出

3. **医療保険証**
   - 健康保険証のコピー（初回申請時）

4. **マイナンバーカード**（または通知カード）
   - 本人確認書類として必要

5. **印鑑**
   - シャチハタ以外の認め印

**準備のポイント：**
- 診察を受けているかかりつけ医師を事前に確認しておく（医師意見書作成に必要）
- 医師意見書は市区町村から医療機関に直接依頼されるケースが多いが、自分で医師に申告する場合もあります
- 訪問看護ステーションから申請サポートを受けられる場合もあるため、ステーション担当者に相談してください

### ステップ2：申請窓口での手続き

**申請場所の選択肢：**

1. **市区町村の介護保険課**（最も一般的）
   - 窓口での対面申請
   - スタッフが記入漏れをその場で確認
   - 営業時間：平日8:30～17:00程度（自治体による）

2. **地域包括支援センター**（65歳以上に限定）
   - 高齢者支援の専門機関
   - より詳しい相談ができる場合がある
   - 訪問看護サービスについても相談可能

3. **福祉事務所**（第2号被保険者の場合がある）
   - 医療保険者での申請対応

4. **訪問看護ステーション**（一部対応）
   - 訪問看護利用者の場合、ステーションが代行申請することもあります
   - 申請書作成をサポートしてくれるケースもあります

**申請時の注意点：**
- 申告者が本人でない場合、委任状が必要な場合があります（自治体により異なる）
- 成年後見人がいる場合、後見人手帳の写しが必要です
- 申請は本人の同意の下で行う必要があります

### ステップ3：訪問調査の受診

申請から数日～1週間以内に、市区町村の認定調査員（ケアマネジャーなど）が自宅を訪問します。

**訪問調査でチェックされる項目（抜粋）：**

| 調査項目カテゴリ | 具体的な内容 |
|-----------------|-----------|
| **身体機能** | ベッドからの起き上がり、歩行速度、転倒リスク、関節可動域 |
| **生活動作** | トイレ使用、入浴、着替え、食事の自立度 |
| **認知機能** | 日時の認識、自分がいる場所の認識、人物認識 |
| **行動障害** | 徘徊、失禁、寝言、室内での危険行動 |
| **社会参加** | 週1回以上の外出、友人との交流、役割行動 |

**調査に向けた準備：**

- **日常の様子を記録しておく**：特に認知症や行動障害がある場合、実際の様子を具体的に伝えることが重要です。例えば「夜間に2～3回起床する」「食後、排便の有無を忘れる」など、具体的な頻度を記録しておくと調査の正確性が高まります

- **本人の体調に配慮**：可能なら、調査者には「本人が疲れやすい」「痛みがある箇所」など、その日の状態を事前に伝えておきます

- **家族の同席**：できるだけ本人のケアに関わっている家族が同席し、訪問看護や医療状況について補足説明するとよいでしょう

- **環境整備**：訪問調査に向けて、住環境を整えすぎないことが大切です。実際の生活環境を見てもらうことで、より正確な判定につながります

**訪問調査の所要時間：**
平均30分～1時間程度（対象者の状態により異なる）

### ステップ4：医師意見書の作成

かかりつけ医が、医学的な観点から要介護度を判定するための意見書を作成します。

**医師意見書に記載される内容：**
- 基礎疾患と現在の治療状況
- 認知機能に関する医学的評価
- 日常生活の制限を生じさせている症状や兆候
- 医学的管理の必要性

**医師意見書作成の流れ：**

```
市区町村 → 医療機関に直接依頼（多くの場合）
          または
本人・家族 → かかりつけ医に申告して記入依頼（自治体により異なる）
```

**費用：** 医師意見書作成費は被保険者負担で、通常は無料です（一部の自治体では保険診療扱い）

**かかりつけ医がいない場合：**
市区町村が指定する医療機関で新たに診察を受ける必要があります。その場合、本人の医療情報を記した紹介状があると効率的です。訪問看護ステーションの看護師に相談して、紹介状の取得をサポートしてもらうこともできます。

### ステップ5：判定と認定の通知

訪問調査と医師意見書をもとに、介護認定審査会（医師、ケアマネジャー、福祉専門職で構成）が判定を行います。

**判定から通知までの期間：**
申請から通常**30日以内**（法定期限は30日）に認定結果が通知されます。ただし、申請が集中している時期には、期間内に完了しない場合もあります。

**認定通知に含まれる情報：**
- 認定区分（要支援1・2、要介護1～5、非該当のいずれか）
- 認定有効期間（原則12ヶ月。再認定までの期間）
- 支給限度額（月額の介護保険利用上限）

**不服がある場合：**
通知受取から60日以内であれば、都道府県の介護保険審査会に異議申し立てができます。

## 要介護度が決まった後｜訪問看護の利用開始

### ケアプラン作成の重要性

認定が決まったら、次は**ケアプラン**（介護サービス計画）を作成します。ケアプランとは、本人と家族のニーズを踏まえ、どの介護サービスをどのくらい利用するかを計画したものです。

**ケアプランの作成主体：**

| 認定区分 | 担当者 | 相談先 |
|---------|--------|------|
| 要支援1・2 | 地域包括支援センター（無料） | 市区町村に確認 |
| 要介護1～5 | 居宅介護支援事業所（ケアマネジャー）/ 施設（入所の場合） | 市区町村または訪問看護ステーション |

### 訪問看護サービスの種類と対象

ケアプランに訪問看護が組み込まれた場合、以下のようなサービスが提供されます：

**医療処置が必要な場合：**
- カテーテル管理、人工肛門ケア、透析シャント管理、褥瘡ケア
- 点滴、注射などの医療処置
- 医療機器（酸素、胃瘻、吸引機）の管理

**健康管理：**
- 血圧・体温などのバイタルサイン測定
- 症状や健康状態の観察と医師への報告

**生活支援と リハビリテーション：**
- 入浴
