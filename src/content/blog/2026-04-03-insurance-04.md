## 在宅医療が「負担になってしまう」前に知っておきたい制度

訪問看護や在宅医療を受け始めた時、多くのご家族が直面する悩みが「思った以上に医療費がかかった」というものです。毎月の訪問看護費用、医薬品代、医療機器のレンタル費用…これらが重なると、家計に大きな負担がのしかかります。

しかし、**ご存知でしょうか？一定以上の医療費を支払った場合、超過分が返ってくる「高額療養費制度」があります**。この制度を正しく理解し活用することで、実際の自己負担を大幅に軽減できるのです。

本記事では、高額療養費制度が訪問看護・在宅医療にどのように適用されるのか、申請方法から計算方法まで、訪問看護の現場経験を踏まえて詳しく解説します。制度の仕組みを理解することで、経済的な不安を減らし、より安心した在宅医療生活へ進むことができます。

---

## 高額療養費制度の基本をおさえよう

### 高額療養費制度とは何か

**高額療養費制度とは、医療機関で支払った医療費が一定額を超えた場合、その超過分が還付される公的な制度**です。日本の医療保険制度では、通常、診療費の3割を患者が自己負担しますが（70歳以上は1〜3割）、この自己負担額に月ごとの上限を設けているのが高額療養費制度の特徴です。

この制度は、高額な治療が必要な患者さんや、長期の入院・通院が必要な方を経済的に守るために存在します。**訪問看護や在宅医療も対象となる医療費であり、条件を満たせば自動的に適用される場合も、申請により返還される場合もあります**。

### 訪問看護は対象になる？

**結論として、訪問看護は保険診療として提供されている場合、高額療養費制度の対象となります**。ただし、これには重要な条件があります：

- 健康保険に加入していること
- 訪問看護が保険診療として認可されていること
- 医師の指示のもと、訪問看護ステーションから提供されていること

なお、自費診療（保険外の特別なサービス）や、医療保険と関係のない福祉サービスは対象外です。訪問看護の利用を開始する際に、必ずケアマネジャー（介護保険の場合）や主治医に「保険診療での提供か」を確認してください。

### 在宅医療全般での適用範囲

高額療養費制度の対象となる在宅医療関連の費用は、以下のとおりです：

| 費用の種類 | 対象となるか | 詳細 |
|---------|---------|------|
| 訪問看護料 | ✅対象 | 医保・介保の訪問看護費 |
| 訪問診療料 | ✅対象 | 医師による在宅診療費 |
| 処方箋薬代 | ✅対象 | 医師の処方による医薬品 |
| 医療用医薬品 | ✅対象 | 訪問看護師が使用する医薬品 |
| 医療用医具 | ⚠️条件付き | 医療保険の対象のみ（介保レンタルは対象外） |
| 医療用ベッド購入 | ❌対象外 | 購入品は対象外（レンタルで保険診療は対象） |
| 福祉用具購入・レンタル | ❌対象外 | 介護保険での費用は対象外 |

**重要な注意：介護保険で提供されるサービス（介護の訪問看護、福祉用具など）は高額療養費制度の対象外です**。医療保険と介護保険の組み合わせで在宅医療を受けている場合、費用が混在しているため、どちらが対象かを事前に確認することが重要です。

## 自己負担限度額の計算方法と具体例

### 自己負担限度額の仕組み

高額療養費制度では、**所得水準と年齢によって、月ごとの自己負担限度額が異なります**。この限度額を超えた分が後日還付される仕組みです。

限度額は「診療月の月初から月末までの医療費」を対象に計算され、**毎月1日にリセット**されます。つまり、1月に100万円の医療費がかかっても、2月に限度額を超えた分を申請すればリセットされ、2月の新しい限度額の計算がスタートするわけです。

### 年齢別・所得別の限度額

以下は、令和6年度現在の標準的な自己負担限度額です：

#### **70歳未満の場合**

| 所得区分 | 月額限度額 | 多数回該当時* | 計算式の目安 |
|---------|---------|---------|---------|
| 年収1,160万円以上 | 252,600円 + （医療費−842,000円）×1% | 140,100円 | |
| 年収770〜1,160万円 | 167,400円 + （医療費−558,000円）×1% | 93,000円 | |
| 年収370〜770万円 | 80,100円 + （医療費−267,000円）×1% | 44,400円 | |
| 年収370万円以下 | 57,600円 | 44,400円 | |
| 生保受給者 | 35,400円 | 24,600円 | |

*「多数回該当」=過去12ヶ月で3回以上高額療養費の支給を受けた場合、4回目以降の限度額

#### **70〜74歳の場合**

| 所得区分 | 月額限度額 | 多数回該当時 |
|---------|---------|---------|
| 現役並み所得者（年収1,160万円以上相当） | 252,600円 + （医療費−842,000円）×1% | 140,100円 |
| 現役並み所得者（年収770〜1,160万円相当） | 167,400円 + （医療費−558,000円）×1% | 93,000円 |
| 現役並み所得者（年収370〜770万円相当） | 80,100円 + （医療費−267,000円）×1% | 44,400円 |
| 一般 | 18,000円 | 8,000円 |
| 低所得Ⅱ | 8,000円 | 8,000円 |
| 低所得Ⅰ | 8,000円 | 8,000円 |

#### **75歳以上（後期高齢者医療）の場合**

| 所得区分 | 月額限度額 | 多数回該当時 |
|---------|---------|---------|
| 現役並み所得者 | 252,600円 + （医療費−842,000円）×1% | 140,100円 |
| 一般 | 18,000円 | 8,000円 |
| 低所得Ⅱ | 8,000円 | 8,000円 |
| 低所得Ⅰ | 8,000円 | 8,000円 |

**各自治体により、さらに自己負担を減らす制度（医療費補助制度など）がある場合もあります。住んでいる地域の役所に確認してください**。

### 実例で学ぶ：在宅医療での計算ケース

#### **ケース1：70歳未満、年収500万円の患者さん**

**1ヶ月の医療費内訳：**
- 訪問診療料：30,000円
- 訪問看護料（週3回、4週間）：60,000円
- 処方箋薬代（3割負担）：12,000円
- 在宅酸素レンタル（保険診療）：8,000円
- **合計医療費：110,000円**
- **患者の自己負担分（3割）：33,000円**

**計算：**
- 該当所得区分：「年収370〜770万円」
- 月額限度額：80,100円 + （医療費－267,000円）×1%
- この患者さんの場合：医療費が110,000円なので限度額計算式は使わず、実際の自己負担33,000円がそのまま適用される
- **実際の自己負担額：33,000円（限度額以下）**
- **還付金額：0円**

この例では、医療費がそこまで高額ではないため、限度額に達せず還付がありません。

#### **ケース2：70歳未満、年収500万円の患者さん（高額なケース）**

**1ヶ月の医療費内訳：**
- 訪問診療料：80,000円
- 訪問看護料（週5回、4週間）：150,000円
- 処方箋薬代（3割負担）：45,000円
- 在宅酸素・人工呼吸器等：75,000円
- **合計医療費：350,000円**
- **患者の自己負担分（3割）：105,000円**

**計算：**
- 該当所得区分：「年収370〜770万円」
- 月額限度額：80,100円 + （医療費－267,000円）×1% = 80,100円 + 830円 = **80,930円**
- **実際の自己負担額：80,930円**
- **還付金額：105,000円 − 80,930円 = 24,070円**

この例では、自己負担105,000円が限度額80,930円を超えているため、差額の24,070円が還付されます。

#### **ケース3：75歳以上、一般所得の患者さん**

**1ヶ月の医療費内訳：**
- 訪問診療料：60,000円
- 訪問看護料（週2回、4週間）：80,000円
- 処方箋薬代：30,000円
- **合計医療費：170,000円**
- **患者の自己負担分（1割）：17,000円**

**計算：**
- 該当所得区分：「一般」
- 月額限度額：18,000円
- **実際の自己負担額：17,000円（限度額以下）**
- **還付金額：0円**

75歳以上は通常1割負担で、自己負担限度額も低めに設定されています。

## 訪問看護利用時の高額療養費申請手続き

### 申請方法の種類と流れ

高額療養費の還付を受けるための申請方法は、**自動的に行われる場合と、自分で申請する必要がある場合**の2パターンがあります。

#### **パターン1：自動還付（被保険者が申請しない場合）**

病院や診療所で治療を受けた場合、医療機関が医療保険者に請求し、**保険者が自動的に受付番号を通知する仕組みがあります**。この場合、被保険者側の申請は不要です。

**訪問看護の場合**も、訪問看護ステーション経由で保険請求が行われるため、通常は自動処理されます。ただし、確実な還付を受けるためには、以下の確認が必要です：

- 健康保険証の記載内容が正確か
- 訪問看護ステーションが正しく保険請求を行ったか
- 保険者からの通知（高額療養費支給決定通知）が届いているか

#### **パターン2：自分で申請する場合**

自動的に還付されない場合や、月の途中で保険が切り替わった場合など、**自分で申請する必要があります**。以下が標準的な申請手続きです：

**申請手続きの流れ：**

1. **必要書類の準備**
   - 高額療養費支給申請書（保険者から提供）
   - 健康保険証（コピー可）
   - 本人確認書類（運転免許証・マイナンバーカードなど）
   - 医療費の領収書
   - 通帳（還付先口座確認用）

2. **申請先の確認**
   - 会社員：健保組合または協会けんぽの支部
   - 自営業・無職：市区町村役所（国民健康保険）
   - 75歳以上：市区町村役所（後期高齢者医療広域連合）

3. **申請書の記入**
   - 被保険者情報（保険証記号・番号、氏名など）
   - 医療機関情報（訪問看護ステーション名、所在地）
   - 治療年月日、医療費額
   - 還付先口座情報

4. **提
