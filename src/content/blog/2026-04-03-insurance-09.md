## 訪問看護のケアプランと居宅サービス計画の関係を正しく理解する

在宅ケアを受けるとき、「ケアプラン」と「居宅サービス計画」という似た言葉を耳にしませんか？両者の違いや関係性がよくわからず、混乱している方も多いはずです。実は、この2つは密接に関連していますが、作成者も役割も異なります。本記事では、訪問看護のケアプランがどのように居宅サービス計画に組み込まれるのか、実務的な流れと具体例をご紹介します。ケアマネジャーや訪問看護師と連携する際の重要なポイントも含めて、あなたの在宅ケアをより良いものにするためのガイドをお届けします。

---

## ケアプランと居宅サービス計画の基本的な違い

### ケアプランとは何か

**ケアプランは、訪問看護事業所が利用者の看護ニーズに基づいて作成する看護計画**です。医療用語では「看護計画」「看護ケア計画」とも呼ばれています。

ケアプランの定義と役割：
- 利用者の健康課題を分析し、看護目標を設定
- 具体的な看護ケアの内容と実施方法を記載
- 訪問看護師が日々のケアに使用する実践的な計画
- 利用者の身体状況の変化に応じて随時更新

訪問看護のケアプランは、以下の医学的背景に基づいて作成されます：

| 要素 | 内容 |
|------|------|
| アセスメント | 利用者の症状、ADL（日常生活動作）、家族背景などの現状把握 |
| 看護診断 | 健康課題の分析（例：「栄養不良のリスク」「排泄管理不全」） |
| 目標設定 | 短期目標（1-2週間）と長期目標（1-3ヶ月）の設定 |
| 実施計画 | 看護介入の具体的な方法と頻度 |
| 評価方法 | ケアの効果測定とフィードバック |

### 居宅サービス計画（介護保険ケアプラン）とは

**居宅サービス計画は、ケアマネジャー（介護支援専門員）が作成する、介護保険によるサービス全体の総合計画**です。介護保険法に基づく法定文書です。

居宅サービス計画の特徴：
- 要介護認定を受けた利用者を対象
- 訪問看護、訪問介護、福祉用具貸与、デイサービスなど、複数サービスを統合
- ケアマネジャーが利用者や家族の希望、課題を聞き取り作成
- 1ヶ月～3ヶ月ごとに見直し（モニタリング）を実施

| 要素 | 内容 |
|------|------|
| 利用者の生活課題（ニーズ） | 「自分らしく生活したい」「家族の負担を減らしたい」など |
| サービス目標 | 複数サービスを組み合わせて達成する目標 |
| サービス内容と頻度 | どのサービスを週に何回、何分利用するか |
| サービス提供者 | 訪問看護事業所、訪問介護事業所など |
| 利用者負担額 | 自己負担額と保険負担額の内訳 |

## 居宅サービス計画とケアプランの作成プロセス

### 居宅サービス計画作成の流れ

**ステップ1：要介護認定申請と判定（介護保険担当窓口）**

在宅ケアを開始する前提として、市町村に要介護認定の申請が必要です。

申請から認定までの流れ：
1. 市町村の介護保険担当窓口に申請書提出
2. 認定調査員が訪問調査（約30分～1時間）
3. 医師の意見書を市町村が取得
4. 介護認定審査会で審査判定
5. 認定結果通知（申請から約30日以内）

認定結果により以下のいずれかが決まります：
- 要介護1～5：介護保険サービスを利用可能
- 要支援1～2：介護予防サービスを利用可能
- 非該当：介護保険対象外（自費サービスなど検討）

**ステップ2：ケアマネジャーとの相談（初回面談）**

要介護認定を受けたら、ケアマネジャーを選びます。相談は無料です。

初回面談での確認事項：
- 利用者の氏名、年齢、生活背景、家族構成
- 現在の健康状態と病歴
- 介護が必要になった主な原因
- 現在利用しているサービス（医療・福祉）
- 生活上の困りごと・希望
- 今後の生活目標

**ステップ3：アセスメントと課題分析**

ケアマネジャーが利用者・家族の話を聞きながら、以下を分析します。

| 分析項目 | 具体例 |
|---------|--------|
| 身体的ニーズ | 排泄介助、入浴介助が必要 |
| 医療的ニーズ | 血圧管理、褥瘡（床ずれ）予防が必要 |
| 心理社会的ニーズ | 社会交流がしたい、独立心を保ちたい |
| 環境的ニーズ | 段差解消、手すり設置が必要 |

**ステップ4：訪問看護事業所との連携協議**

医療ニーズが高い場合、ケアマネジャーは訪問看護事業所と連携します。

この段階での情報共有：
- 利用者の主治医や現在の医療状況
- 介護保険で対応できる支援内容
- 医療保険で対応すべき看護ニーズ
- 訪問看護の利用予定（頻度、時間）

**ステップ5：居宅サービス計画（第1号様式）の作成**

ケアマネジャーが、利用者や家族の要望を踏まえて、サービス内容を決定します。

計画書に記載される必須項目：
1. 利用者の生活課題（ニーズ）
2. サービス目標（短期・長期）
3. サービス内容（訪問看護、訪問介護など）
4. 各サービスの頻度・時間
5. 各サービス提供者の名称
6. サービス開始予定日
7. 計画期間
8. 利用者負担額

### 訪問看護ケアプラン作成の流れ

**ステップ1：訪問看護事業所への利用申し込み**

居宅サービス計画に訪問看護が含まれたら、利用者が直接または紹介で申し込みます。

申し込み時の確認事項：
- 利用者の生年月日、連絡先
- 訪問看護の利用保険（介護保険 or 医療保険）
- 主治医の氏名と医療機関
- 現在の病状・症状
- 介護者の状況
- 自宅の住所と施設構造

**ステップ2：初回訪問とアセスメント**

訪問看護師が初回訪問で、以下をアセスメント（情報収集と分析）します。

訪問看護師が確認するアセスメント項目：

| カテゴリ | 確認項目 | 具体例 |
|----------|---------|--------|
| 身体機能 | ADL（移動、排泄、食事） | 車いす使用、つかまり立ち可能か |
| 認知機能 | 見当識、意思疎通 | 日付がわかるか、指示理解があるか |
| 心理状態 | 気分、適応状態 | 不安はないか、家に慣れているか |
| 生活環境 | 住居形態、危険要因 | 転倒リスク、温度管理、感染予防 |
| 医療状況 | 病名、治療内容、医薬品 | 糖尿病、褥瘡、使用薬剤 |
| 家族背景 | 家族構成、介護力 | 同居家族の有無、仕事の有無 |

**ステップ3：主治医への指示確認**

訪問看護開始前に、主治医（かかりつけ医）から訪問看護指示書（医療保険）またはケアマネジャーを通じての情報を得ます。

医師からの指示に含まれる内容：
- 訪問看護の必要性と期間
- 実施すべき看護内容
- 症状悪化時の対応方法
- 他職種との連携内容
- 医学的禁忌事項

**ステップ4：看護ケアプランの作成**

初回訪問のアセスメント結果と主治医指示に基づいて、訪問看護師が看護ケアプランを作成します。

看護ケアプランに記載される主要項目：

1. **看護目標の設定**
   - 短期目標（1～2週間）：「血圧が130/80以下に保たれる」
   - 長期目標（4週間～）：「自分で薬管理ができるようになる」

2. **看護問題の抽出**
   - NANDA-I看護診断を使用（国際的な診断体系）
   - 例：「栄養不良のリスク」「自己ケア不足」

3. **看護介入（ケア内容）**
   - バイタルサイン測定（血圧・脈拍・体温）
   - 褥瘡予防ケア
   - 排泄管理
   - 服薬管理
   - 栄養管理指導
   - リハビリテーション支援
   - 精神心理的サポート

4. **実施方法と頻度**
   - 「週2回火木曜日、午前10時に訪問」
   - 「1回の訪問時間：40分」
   - 「担当看護師：〇〇看護師、〇〇看護師」

5. **観察項目と評価方法**
   - 毎回測定するデータ
   - 月1回のアセスメント見直し
   - 目標達成度の判定基準

**ステップ5：利用者・家族への説明と同意**

作成したケアプランを利用者や家族に説明し、同意を得ます。

説明時に確認される事項：
- ケアの内容と目的
- 訪問日時と担当者
- 料金（自己負担額など）
- 緊急時の連絡方法
- プライバシーの保護
- 苦情・相談窓口

## 訪問看護ケアプランの必須項目と作成の実践ポイント

### 訪問看護ケアプランに含まれる必須要素

訪問看護のケアプランは、以下の要素を含んだ実践的な計画です（訪問看護療養費基準に基づく）。

**1. 利用者の基本情報**

|------|--------|
| 氏名・生年月日 | 田中花子 1945年3月15日生（78歳） |
| 住所・電話番号 | ○○県○○市○○町1-1-1
