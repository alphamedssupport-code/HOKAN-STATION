## イントロダクション

「訪問看護の料金が急に高くなった」「加算って何が対象なの？」——在宅医療の現場では、加算制度についての疑問や不安がよく聞かれます。実は訪問看護の料金は、基本的な訪問看護料だけでなく、患者さんの状態や提供するサービスに応じた**複数の加算**で成り立っています。緊急対応が必要な患者さん、人生の最期を自宅で迎えたいターミナル期の患者さん、複雑な医療処置が必要な患者さんなど——様々なニーズに応えるための仕組みがこの加算制度です。本記事では、訪問看護の主要な加算制度について、算定要件・金額・実例を具体的に解説し、患者さんと家族が納得できる在宅医療の計画立案をお手伝いします。

## 訪問看護の加算制度の基本構造

### 加算制度とは何か

訪問看護の料金体系は、**基本となる訪問看護料**に対して、患者さんの状態や提供する追加的なサービスに応じて**加算（上乗せ料金）**が付与される仕組みです。この加算制度の目的は、複雑な医療ニーズを抱える患者さんへの対応や、24時間体制での緊急対応など、通常以上のコストがかかるケースに対して、事業所の経営を支えつつ、質の高いサービスを提供できるようにすることです。

加算は大きく分けて、以下のカテゴリに分類されます。

- **患者さんの状態に応じた加算**：特定の疾患や状態がある場合に算定
- **サービス提供体制に関する加算**：24時間対応体制や緊急時対応などの運営方針に基づいて算定
- **人員配置や資格に関する加算**：特定の資格を持つ職員による対応時に算定

加算の種類や要件は、**介護保険と医療保険で異なる**ため、患者さんがどちらの保険で訪問看護を受けているかを確認することが重要です。

### 介護保険と医療保険での加算の違い

訪問看護は介護保険と医療保険の両制度で提供されており、**加算の種類・金額・要件が異なります**。

| 項目 | 介護保険 | 医療保険 |
|------|---------|---------|
| **対象者** | 要介護1～5、要支援1～2 | 65歳未満、または介護保険給付対象外の医療ニーズが高い者 |
| **加算の種類** | 約10種類 | 約15種類 |
| **月額料金体系** | 月の訪問回数で基本料金が決まり、加算が加わる | 訪問1回ごとに料金が決まり、加算が加わる |
| **自己負担** | 介護保険の対象は1割～3割、加算も同率 | 医療保険は原則3割（高額療養費制度の対象） |

患者さんが受け取る**訪問看護計画書や領収書に「加算」の項目が記載されている場合**は、どの加算がどの理由で算定されているかを確認することが重要です。わからない場合は、訪問看護ステーションに問い合わせてください。

## 介護保険の主要加算制度

### 1. 緊急時訪問看護加算

#### 加算の内容と算定要件

**緊急時訪問看護加算**は、患者さんが急変した場合に**24時間以内に訪問看護を行う体制を整えている事業所**が算定できる加算です。月額で加算され、以下の要件をすべて満たす必要があります。

**算定要件：**
- 訪問看護計画書に「緊急時対応の体制がある」と明記されている
- 24時間体制（待機体制）で緊急対応が可能である
- 患者さん（または家族）が緊急時の連絡先を把握している
- 前年度の算定月において、実際に緊急訪問を行った実績がある（要件としての解釈は事業所で異なる場合があり、詳細は各都道府県の通知を確認）

**加算額（2024年度現在）：**
- 月額 **450円**（介護保険、1割負担なら45円の自己負担）

#### 実際の活用例

**事例：脳梗塞の後遺症で日中一人の患者さん**

80歳の患者さんが脳梗塞の後遺症で左半身麻痺がある場合、日中に子どもが仕事で留守にすることが多いため、万が一の急変時に対応できるよう緊急時訪問看護加算を算定します。実際に、ある夜間に患者さんが転倒した際、訪問看護ステーションが2時間以内に対応し、救急車の手配や初期対応を行った実例があります。

このような実例を通じて、患者さんと家族に「加算は単なる料金ではなく、生命を守るための体制費である」ことを理解してもらうことが重要です。

#### 緊急時訪問看護加算の流れ（チェックリスト）

患者さんが緊急時訪問看護加算の対象かどうかを判断するためのチェックリストです。

- [ ] 患者さんが24時間体制での対応が必要な状態か（例：化学療法中、呼吸管理が必要、状態が不安定）
- [ ] 訪問看護ステーションが24時間対応体制を整えているか
- [ ] 患者さん・家族が緊急時の連絡方法を理解し、同意しているか
- [ ] 家族が日中不在など、実際に一人の時間が多いか
- [ ] 主治医が緊急時対応の必要性を認めているか

3つ以上当てはまる場合は、訪問看護ステーションに相談し、加算の対象となるかどうか確認することをお勧めします。

### 2. ターミナルケア加算

#### 加算の内容と意義

**ターミナルケア加算**（終末期加算）は、患者さんが**人生の最期を自宅で迎える際に、身体的・心理的・社会的・スピリチュアルなケアを提供する加算**です。これは単なる「看護料」ではなく、患者さんと家族が納得できる最期を実現するための支援体制を評価した加算です。

**算定要件：**
- 主治医から「余命6ヶ月以内」と診断されている（医学的根拠あり）
- 訪問看護計画書に「ターミナルケア」として明記されている
- 患者さん・家族が在宅でのターミナルケアを希望している（同意文書あり）
- 訪問看護ステーションが24時間体制でのサポートを提供できる
- 医師、訪問看護師、介護職など多職種での連携が成立している

**加算額（2024年度現在）：**
- **1回の訪問ごとに 2,000～3,000円**（介護保険の場合。医療保険の場合はさらに高額）
- 月間の訪問回数に応じて累積される
- **死亡日当日の訪問には さらに 5,000～10,000円の加算**（死亡診断補助料、ご遺族への対応費など）

#### ターミナルケアの具体的な内容

ターミナルケア加算が算定される期間に、訪問看護師が提供するサービスの例：

| サービス内容 | 具体的な対応 |
|-----------|-----------|
| **身体的ケア** | 疼痛管理（麻薬性鎮痛薬の管理・調整）、呼吸困難の緩和、排泄コントロール、スキンケア |
| **症状緩和** | 不安、吐き気、倦怠感などの症状に対する対応 |
| **家族支援** | 介護方法の指導、死後の準備の説明、グリーフケア（喪失感への対応） |
| **医師・薬剤師との連携** | 処方薬の見直し、新たな症状への対応、24時間の相談体制 |
| **スピリチュアルケア** | 本人・家族の価値観や希望を尊重した最期の時間作り |
| **ご遺族対応** | 故人のケア、ご遺族への心理的サポート、市役所手続きの案内 |

#### 実例：がん患者さんの自宅でのターミナルケア

**事例：進行がんで自宅療養を希望された75歳男性**

患者さんは「最期は自分の部屋で、家族に見守られたい」という希望を持っていました。訪問看護ステーションは、以下のようにサポートを提供しました。

1. **初期段階（診断から3ヶ月）**：疼痛管理の強化、本人・家族への心理的支援
2. **中盤（最後の1ヶ月）**：週4～5回の訪問、家族に介護方法を教える、医師との連絡調整を密に
3. **最終段階（最後の1週間）**：ほぼ毎日訪問、症状変化への即座の対応、家族のグリーフケア
4. **死後対応**：ご遺族への支援、市役所手続きの案内

この事例では、ターミナルケア加算によって訪問頻度を増やし、患者さんと家族の希望を実現することができました。

### 3. 専門的ケア加算

#### 加算の概要と対象となる医療処置

**専門的ケア加算**は、特定の高度な医療処置が必要な患者さんに対応する訪問看護師の**専門知識と技術を評価する加算**です。この加算があることで、事業所は追加の研修・機器購入・ベテラン職員の配置を実現しており、患者さんはより安全で質の高いケアを受けられます。

**対象となる医療処置（主なもの）：**

| 医療処置 | 具体的な内容 |
|--------|-----------|
| **呼吸管理** | 人工呼吸器の管理、気管切開の管理、酸素療法 |
| **栄養管理** | 中心静脈栄養（CVカテーテル）の管理、胃ろう・腸ろうの管理 |
| **排泄管理** | 尿路カテーテルの管理、膀胱留置カテーテル交換 |
| **創傷管理** | 褥瘡（床ずれ）の処置、術後創部の管理、難治性潰瘍の処置 |
| **疼痛管理** | 麻薬性鎮痛薬の自己注射、脊髄くも膜下鎮痛薬の管理 |
| **点滴管理** | 抗がん剤投与、抗生物質の継続投与 |
| **人工肛門・人工膀胱** | 装具交換、皮膚のケア |

**加算額（2024年度現在）：**
- **月額 1,500～3,000円**（処置の種類・複雑度に応じて変動）
- 複数の処置がある場合は、加算が合算される場合と、最高額で頭打ちになる場合がある（制度の詳細は事業所に確認）

#### 専門的ケア加算の実例

**事例1：人工呼吸器が必要な患者さん**

60歳の筋萎縮性側索硬化症（ALS）患者さんが、自宅で人工呼吸器を使いながら生活を続けたいと希望した場合、訪問看護師は以下の対応をします。

- 毎日の人工呼吸器の動作確認、バッテリー管理
- 気道の吸引（分泌物除去）
- 気管切開部の清潔保持と処置
- 急変時の対応（バッグバルブマスク手動換気など）
- 家族への教育・サポート

この複雑で生命を左右する処置に対応するため、**専門的ケア加算が月額3,000円**算定されます。この加算によって、訪問看護ステーションは呼吸器の研修を受けた専門職を配置し、24時間対応の体制を整えることができます。

**事例
