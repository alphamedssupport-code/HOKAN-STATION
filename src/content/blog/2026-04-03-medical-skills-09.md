## イントロダクション

「患者さんが『痛い』と言っているけど、どれくらいの痛みなのか判断がつかない」「疼痛評価ツールを使っているものの、正しく使えているのか不安」——こうした悩みは、訪問看護師であれば誰もが経験するものです。

在宅医療の現場では、患者さんが医師の前でしか診察を受けないため、**訪問看護師による正確な疼痛評価が医療判断の重要な情報源**になります。NRS（数値評価スケール）、VAS（視覚的アナログスケール）、フェイススケール（顔スケール）といった疼痛評価ツールは、患者さんの痛みを「見える化」し、より適切なケアや医療調整につなぐための不可欠なツールです。

本記事では、各評価ツールの特徴、正しい使い方、実践的な応用方法、そして評価結果の記録・報告までを、訪問看護の現場経験に基づいて解説します。読み終えた後には、患者さんの痛みをより正確に把握し、医師との情報共有をスムーズに行えるようになります。

## 疼痛評価がなぜ重要なのか

### 訪問看護における疼痛評価の役割

疼痛（痛み）は極めて主観的な症状です。医学教科書でよく引用される定義は、**「実際の組織損傷がなくても、患者さんが『痛い』と感じていればそれが痛みである」**というものです。

訪問看護では、患者さんと医師が頻繁に接する医療機関とは異なり、患者さんの痛みの変化や日常生活への影響を最も直近で観察する立場にあります。正確な疼痛評価は以下の点で極めて重要です：

- **医師への情報提供**：訪問診療医や担当医が在宅患者さんの症状を把握する主要な情報源となる
- **治療調整の根拠**：鎮痛薬の種類・用量の変更が必要かどうかの判断材料
- **患者教育の基準**：痛みの変化を数値で可視化することで、患者さん自身の健康管理意識が高まる
- **ケアプランの立案**：痛みが日常生活動作（ADL）に与える影響を客観的に評価できる

### 疼痛評価ツール導入のガイドライン

**日本がん支持治療学会**が発表した「がんの痛み治療ガイドライン」では、患者さんの痛みを適切に評価することを医療の質向上の要件としており、以下のガイドラインが示されています：

- 痛みの有無・程度を定期的に評価することが推奨される
- 患者さんのコミュニケーション能力に応じた複数の評価ツールの選択が重要
- 同じ患者さんには、同じ評価ツールを継続使用して経時変化を追跡すべき

訪問看護の現場では、単に痛みの数値を記録するのではなく、**その背景にある身体的・心理的・社会的因子を総合的に評価する姿勢**が求められます。

## NRS（数値評価スケール）の使い方

### NRSとは何か

**NRS（Numeric Rating Scale）**は、0～10の数字で痛みの強さを表す評価ツールです。国際疼痛学会（IASP）でも認可されており、世界的に最も広く使われている疼痛評価ツールの一つです。

| 数値 | 痛みの状態 | 臨床的な解釈 |
|------|-----------|-----------|
| 0 | 痛みなし | 痛みがない状態 |
| 1～3 | 軽度 | 生活に支障がほぼない |
| 4～6 | 中等度 | 生活に支障あり、治療が必要 |
| 7～9 | 高度 | 日常生活が困難、緊急対応が必要な場合もある |
| 10 | 最高度の痛み | 耐えがたい痛み |

### NRSの正しい使い方（ステップバイステップ）

**ステップ1：準備と環境整備**
- 患者さんがリラックスできる環境を整える（騒音や中断がない場所）
- 複数の評価ツールと混同しないよう、NRS用の説明用紙またはイラストを用意する
- 必要に応じて眼鏡や補聴器を装用してもらう

**ステップ2：患者さんへの説明**
以下のように説明することを推奨します：

「今から、あなたの痛みの強さを数字で表してもらいます。0は『痛みがない』という意味で、10は『これ以上ない最高の痛み』という意味です。あなたの今の痛みは、この0から10の間のどこにあたりますか？」

**ステップ3：患者さんの回答を聞く**
- 患者さんが自発的に数字を言うまで待つ（誘導的な質問は避ける）
- 「4ですか、5ですか？」のような選択肢提示は避ける
- 患者さんが迷っている場合は、「大体でいいですよ」と伝える

**ステップ4：追加情報の聴取**
- 「その痛みはいつ頃から始まりましたか？」
- 「痛みが強くなるときはどんなときですか？」
- 「現在、痛み止めの薬は効いていますか？」

といった補足質問を行い、痛みの背景を理解する

**ステップ5：記録**
カルテに以下の情報を記載します：

```
【疼痛評価】
評価日時：2026年4月3日 14:30
評価ツール：NRS
痛みスコア：6点
痛みの部位：左下肢
痛みの性質：しびれるような痛み
増悪因子：歩行時に増強
緩和因子：安静時はやや軽減
現在の鎮痛薬：アセトアミノフェン500mg 1日3回内服中
患者さんの訴え：「最近、痛みが少し強くなった気がします」
```

### NRS使用時の注意点

**認知機能が低下している患者さんの場合**
- NRSの理解が困難な場合がある
- 簡潔な説明を何度か繰り返す、または別の評価ツール（フェイススケール）の使用を検討

**回答の一貫性が低い場合**
- 「先ほど『5』と言われていたのに、今は『2』と言われていますね。何か変わったことはありますか？」と確認
- 認知機能の変化、新しい薬の開始、精神的ストレスなどがないか聴取

**繰り返し評価の重要性**
- NRSは「その時点での気分や心理状態」に影響を受けやすい
- **定期的（1～2週間ごと）な繰り返し評価が重要**で、1回の評価値だけで判断してはいけない

## VAS（視覚的アナログスケール）の使い方

### VASとは何か

**VAS（Visual Analog Scale）**は、10cm（100mm）の横一本の線で表現された評価ツールです。左端を「痛みなし」、右端を「耐えがたい痛み」として、患者さんが現在の痛みの位置に印をつけます。

```
痛みなし ├─────────────────────┤ 最高の痛み
        0                    100（mm）
```

VASの最大の特徴は、**連続的で段階性がない評価が可能**なため、細微な痛みの変化を検出しやすい点です。臨床試験や研究データでよく使われるのはこの理由によります。

### VASの正しい使い方（ステップバイステップ）

**ステップ1：VAS用紙の準備**
- A4サイズの白い紙に、10cm（100mm）のまっすぐな線を引く
- 左端に「痛みなし」、右端に「我慢できない痛み」と記載
- 患者さんが記入しやすいよう、ペンと定規を用意
- 高齢患者さんには、より大きく見やすいサイズ（A3など）の使用も考慮

**ステップ2：患者さんへの説明**

「この線の左側は『痛みがない』という意味で、右側は『これ以上ない耐えがたい痛み』という意味です。あなたの今の痛みの強さが、この線のどこにあたるか、その場所に『×』の印をつけてください。」

**ステップ3：患者さんに線上に印をつけてもらう**
- 患者さんが自分で印をつけられない場合は、患者さんが示した位置に看護師が代わりにつける
- 「正解」「不正解」という概念がないことを患者さんに伝える

**ステップ4：距離を測定して数値化**
- 定規を使って、左端から印まで測定する
- mm単位で記録する（例：43mm = 4.3点）
- NRSとの比較を容易にするため、10で割って0～10スケールに変換することもある

**ステップ5：記録**

```
【疼痛評価】
評価日時：2026年4月3日 14:30
評価ツール：VAS
痛みスコア：52mm（5.2点相当）
その他の所見：患者さんは印をつける際にやや躊躇されていた
```

### VAS使用時の特徴と注意点

**VASが有用な場面**
- 痛みの微妙な変化を評価したい場合
- 鎮痛薬の効果判定を細かく追跡したい場合
- 研究や臨床試験での評価

**VAS使用時の課題と対策**

| 課題 | 対策 |
|------|------|
| 高齢患者や認知機能低下患者が理解しにくい | NRSやフェイススケールへの変更を検討 |
| 視力低下で線が見えない | 拡大版の用紙を用意 |
| 手の動きに障害がある（脳卒中後遺症など） | 音声での回答またはその他ツールの選択 |
| 毎回、印の位置が大きく異なる | 患者さんの気分や心理状態に左右されやすい可能性→聴取を深める |

**複数回測定時の注意**
- 初回測定時と同じ評価者・同じ時間帯・同じ条件での測定が理想的
- 別の評価者が測定する場合は、患者さんに「異なる看護師が評価する場合があります」と事前に説明

## フェイススケール（顔スケール）の使い方

### フェイススケールとは何か

**フェイススケール（Faces Pain Scale）**は、表情の異なる複数の顔を並べたもので、患者さんがどの顔の表情が自分の気持ちに最も近いかを選ぶ評価ツールです。言語能力に依存しないため、**小児、高齢者、認知機能低下患者、言語障害患者**など、NRSやVASの理解が困難な患者さんに特に有用です。

典型的なフェイススケールは6段階～10段階のものがあります：

**6段階版**
```
😄 😊 😐 😕 😟 😢
0  1  2  3  4  5
```

- 😄（0点）：痛みなし、とても機嫌がいい
- 😊（1点）：少し痛い
- 😐（2点）：中くらいの痛み
- 😕（3点）：かなり痛い
- 😟（4点）：とても痛い
- 😢（5点）：最高の痛み

**国際版**としては、**FPS-R（Faces Pain Scale – Revised）**が WHO（世界保健機関）の推奨評価ツールとなっています。

### フェイススケールの正しい使い方（ステップバイステップ）

**ステップ1：ツールの準備**
- フェイススケールが印刷された紙（A4サイズ）またはタブレット端末
- 色使いが自然で、年齢に適した表情のものを選択
- 小児向けと成人向けで、若干デザインが異なる場合がある

**ステップ2：患者さんへの説明**

「これらの顔は、人が痛みを感
