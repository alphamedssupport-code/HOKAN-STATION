## イントロダクション

「腕や脚が徐々に腫れてきて、着ていた洋服が入らなくなった」「がんの手術後、その側の腕がむくんで困っている」——このような悩みを抱える方は少なくありません。これは**リンパ浮腫**という症状で、日本ではがん患者さんの約5～10%が経験するとも言われています。

リンパ浮腫は完全に治すことが難しい慢性疾患ですが、適切なケアで進行を止め、症状を大幅に改善することは十分に可能です。その中でも「用手的リンパドレナージ（Manual Lymph Drainage, MLD）」は、リンパ液の流れを促進する有効な技術として、訪問看護現場で広く実施されています。

本記事では、用手的リンパドレナージの仕組みから実際の手技、セルフケアのポイントまで、現場の知見をもとに詳しく解説します。自宅でのケアを工夫することで、生活の質を大きく向上させることができます。

## リンパ浮腫とは｜医学的背景の理解

### リンパ浮腫が起こる仕組み

**リンパ浮腫**（lymphedema）は、リンパ液の流れが悪くなり、組織間に液体が溜まった状態です。人間の体には血管のほかに、リンパ管という細い管が張り巡らされており、組織間の余分な液体（リンパ液）を集めて静脈に戻す働きをしています。

リンパ液の流れが障害される原因は大きく2つに分けられます。

**一次性リンパ浮腫**：先天的にリンパ管が発育不全、または欠損している状態。全リンパ浮腫患者の約10～15%を占めます。

**二次性リンパ浮腫**：外傷や手術、感染症、腫瘍などが原因でリンパ管やリンパ節が損傷されたもの。特にがん治療（乳がん、婦人科がん、黒色腫など）に伴うリンパ節郭清（廓清）後に発症する患者さんが最も多く、全体の85～90%を占めます。

特にがん患者さんの場合、術後1年以内に発症することが多く、症状が出ていなくても予防的ケアが推奨されています。

### リンパ浮腫の段階分類（ISL分類）

国際リンパ学会（International Society of Lymphology, ISL）は、リンパ浮腫の重症度を4段階に分類しています。

| **段階** | **症状** | **可逆性** |
|---------|---------|----------|
| **リスク段階** | リンパ節廓清後だが、まだ浮腫がない状態 | 完全可逆 |
| **第I期（軽度）** | 圧迫すると指の跡が残る柔らかい浮腫 | 完全可逆 |
| **第II期（中等度）** | 圧迫しても跡が残りにくい硬い浮腫。繰り返す感染あり | 部分的可逆 |
| **第III期（重度）** | 象皮様の皮膚変化。感染や皮膚トラブルが頻繁 | 不可逆的 |

訪問看護では特に第I～II期の患者さんに対して、進行を食い止めるための早期介入が重要です。

### 日本での統計データ

- **乳がん患者**：リンパ節郭清後、5～10年で15～25%がリンパ浮腫を発症
- **子宮がん患者**：術後のリンパ浮腫発症率は約15～40%
- **黒色腫患者**：リンパ節廓清後の発症率は約15～20%

早期発見・早期ケアにより、進行を遅延させ、生活の質の低下を最小限に抑えることが可能です。

## 用手的リンパドレナージ（MLD）の原理と効果

### MLDの作用機序

用手的リンパドレナージは、**特定の圧と技法でリンパ管を刺激し、リンパ液の流れを促進する手技**です。通常のマッサージとは異なり、リンパの生理学的特性に基づいた科学的なアプローチです。

**リンパ管の特徴：**
- リンパ管の壁には平滑筋があり、収縮することでリンパ液を自動的に流す力がある（リンパポンプ作用）
- この収縮は主に呼吸、筋肉の動き、そして外からの圧刺激により活性化する
- リンパ管の弁機構があり、一方向への流れを支持する

MLDは、この自動的なリンパポンプ作用を最大限に引き出すために、**リズミカルで優しい圧**を加え、近位のリンパ節に向かって段階的にリンパ液を導きます。

### エビデンスに基づいた効果

複数の臨床試験により、MLDの有効性が実証されています。

**主な研究知見：**
- **圧迫療法との組み合わせ**：MLDと圧迫包帯を組み合わせることで、単独使用より約1.5倍の浮腫軽減効果（Badger et al., 2004）
- **周囲径の減少**：4週間の集中的MLD施行により、上肢浮腫では平均5～15cm²の体積減少を報告している研究が多数存在
- **症状緩和**：痛み、重感、動きの制限が有意に改善される患者が75～80%程度

ただし、リンパ浮腫は慢性疾患であり、一度改善しても**継続的なセルフケアなしに再発することが多い**点に注意が必要です。訪問看護では、患者さんの自己管理能力を高めることが治療の成功を左右します。

### MLDの限界と位置づけ

MLDは有効な手技ですが、万能ではありません。

- **完全治癒は難しい**：進行したリンパ浮腫（第III期）では、皮膚の繊維化が進み、可逆的な改善は限定的
- **継続が必須**：やめると数週間で元の状態に戻ることが多い
- **適応と禁忌がある**：感染症、悪性腫瘍の活動期、深部静脈血栓症などではMLDは禁忌とされる

医師の診断と指示のもとで、他のケア（圧迫療法、運動療法など）と組み合わせることが重要です。

## 用手的リンパドレナージの手技と実践方法

### MLDの基本的な手技原則

訪問看護で実施されるMLDには、いくつかの国際的な手技体系があります。その中でも**Vodder法**と**Leduc法**が広く採用されています。

**共通する原則：**

1. **圧は軽い**：通常のマッサージの1/10程度（約30gの圧力）。皮膚の上の組織を動かすイメージ
2. **皮膚をスライドさせる**：摩擦を避け、皮膚表層のリンパ管を刺激する
3. **リズミカルな動き**：1秒かけてゆっくり圧を加え、1秒かけてゆっくり戻す（約1秒1サイクル）
4. **近位方向へ**：常に心臓に向かってリンパを流す方向性を意識

### Vodder法による基本手技

Vodder法は、オーストリアで開発された、最も広く普及している手技です。

**基本的なグリップ（握り方）：**

```
小指と薬指を曲げ、中指と人差し指を伸ばした状態で、
両手の指の丸みを使ってリンパ管を刺激する。
通常のマッサージのように爪を立てたり、指を関節させない。
```

**上肢への施行例（乳がん術後のリンパ浮腫を想定）**

**Step 1: リンパ節の準備（首・腋の下）**

1. 患者さんは仰臥位（あおむけ）または半座位で、リラックスした姿勢をとる
2. 施術者は対側に立ち、患者さんの首の側面に手を当てる
3. リンパ節（頸部リンパ節）に向かい、軽い圧でゆっくり4～6回スライドさせる
4. 次に腋の下（腋窩リンパ節）に手を当て、同様に4～6回施行

**目的**：リンパ液の受け入れ先を先に準備し、流れやすい環境を整える

**Step 2: 上肢遠位部から近位部へのドレナージ**

1. 患者さんの手（手指）から始める
2. 手背（手の甲）全面に両手の指を軽く当て、手首に向かってスライド（4～6回）
3. 前腕へ移動：前腕の手側から肘に向かってドレナージ（複数ラインで4～6回ずつ）
4. 上腕へ移動：上腕全体（内側・外側・後ろ側）を肩に向かってドレナージ

**各ラインでのストローク数**：原則として4～8回。患者さんの反応や浮腫の状態により調整。

**Step 3: 腋窩リンパ節への最終導入**

1. 肩付近から腋の下（腋窩）に向かってドレナージ
2. リンパ液が十分に流入したことを確認

**全体の所要時間**：一側上肢で約30～45分。両側の場合は60～90分。

### Leduc法による手技

Leduc法はベルギーで開発され、より体系的な解剖学的区分を用います。

**特徴：**
- リンパ路を細かく分けて施行
- ポンピングアクション（押して離す）を強調
- 浮腫が著明な部位への施行に適している

**基本的なポンピングアクション：**

```
1. 指を軽く曲げた状態でリンパ管に接する
2. ゆっくり圧を加えながら皮膚をスライド（1秒）
3. ゆっくり圧を抜きながら手をリセット（1秒）
4. 新しい位置で繰り返す
```

訪問看護の現場では、患者さんの浮腫の程度や反応により、Vodder法とLeduc法の要素を適宜組み合わせることが多いです。

### MLDの安全な施行にあたっての確認事項

**各回の施行前のチェックリスト：**

- [ ] 患者さんの医学的禁忌（感染症、活動性悪性腫瘍など）を確認した
- [ ] 患者さんが痛みや違和感を感じていないか、事前に問診した
- [ ] 浮腫部位の皮膚状態を視診した（傷、感染兆候がないか）
- [ ] 患者さんが温かく、快適な環境で施行する（冷えはリンパポンプを低下させる）
- [ ] 施行者の手が清潔で、爪が短く整えられている
- [ ] 患者さんがリラックスしている状態を確認した

**禁忌の確認：**

| **禁忌の状態** | **理由** |
|---|---|
| 感染症（蜂窩織炎など）の急性期 | リンパドレナージにより感染が全身に広がる可能性 |
| 活動性の悪性腫瘍 | 腫瘍細胞の転移を促進する可能性 |
| 深部静脈血栓症（DVT） | 血栓を遊離させるリスク |
| 心不全の急性増悪期 | 循環負荷増加のリスク |
| 未診断の腫脹 | 原因不明の浮腫は医学的評価後に対応 |

これらに該当する場合は、必ず主治医に相談し、MLDの適応を再評価してください。

## 訪問看護におけるMLD実施の実際

### 訪問時間・頻度・期間の目安

**集中治療期（初期）**
- **頻度**：週2～3回、4～8週間
- **所要時間**：1回あたり60～90分
- **目的**：浮腫の体積を
