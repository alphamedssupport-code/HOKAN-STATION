## イントロダクション

精神疾患を抱える患者さんの自宅での療養を支援する訪問看護は、特に服薬管理と危機介入が重要な役割を占めます。統合失調症やうつ病などの患者さんは、環境変化や疲労により症状が悪化することがあり、時には自殺念慮（じさつねんりょ）などの危機的状況に陥ることも少なくありません。多くの訪問看護師は「どのような声かけをすれば信頼関係を築けるのか」「服薬を拒否する患者さんにどう対応するのか」といった課題を抱えています。本記事では、精神科訪問看護の現場で実践できる服薬管理技法と危機介入の具体的スキル、そして患者さんと家族をどう支援するかについて、根拠に基づいて解説します。

## 1. 精神科訪問看護の現状と課題

### 1-1. 訪問看護における精神疾患患者の実態

日本において、精神疾患を持つ患者数は約392万人（2017年患者調査）にのぼり、そのうち在宅療養を選択する患者は年々増加しています。訪問看護ステーションの機能強化加算の取得要件として、精神科訪問看護の提供が重視されるようになり、多くのステーションが対応を拡大しています。

**精神疾患患者が直面する主な課題：**
- 服薬アドヒアランスの低下（40～50%の患者が自己判断で服用を中止）
- 社会的孤立による抑うつ状態の悪化
- 自殺リスクの増加（うつ病患者の自殺企図率：約15～20%）
- 家族による介護負担の増大
- 外出困難による生活機能の低下

### 1-2. 訪問看護師が身につけるべきコア・コンピテンシー

精神科訪問看護の実践には、一般的な看護スキルに加えて以下の専門知識が必要です：

| スキル領域 | 具体的内容 |
|-----------|---------|
| 精神症状のアセスメント | 幻聴・妄想・抑うつ症状の把握 |
| 服薬知識 | 向精神薬の作用・副作用・相互作用 |
| 心理社会的支援 | ラポール形成・傾聴・心理教育 |
| 危機対応 | 自殺リスク評価・安全環境構築 |
| 家族支援 | 心理教育・ストレスマネジメント |

## 2. 服薬管理の実践的スキル

### 2-1. 服薬アドヒアランス低下の原因分析

**医学的要因：**
- 薬の副作用（体重増加、手の震え、性機能障害など）
- 効果を実感できていない
- 多剤併用による飲み忘れ

**心理社会的要因：**
- 「薬に頼りたくない」という心理的抵抗
- 疾病認識の欠如（自分が病気だと認識していない）
- 服薬の必要性を理解していない
- 経済的理由

**環境的要因：**
- 処方箋の入手困難
- 定期受診の中断
- 家族の関心低下

### 2-2. アセスメントの実施手順

訪問初回時から継続的に実施する服薬アセスメントの標準手順を以下に示します。

**STEP 1：服薬状況の確認（初回～毎回）**

```
☐ 現在の処方薬をすべてリストアップ
  （患者さん・家族に確認、処方箋・薬袋を見る）
☐ 実際の服薬状況を確認
  （「毎日飲めていますか」では不十分）
☐ 飲み忘れ・自己中断の有無を確認
☐ 薬の保管状況を確認
  （湿度・光・温度が適切か）
☐ 副作用の有無を聞き取り
  （症状・出現時期・程度を記録）
```

**具体的な聞き取り例：**

> 「〇〇さん、いつもお薬ありがとうございます。今日は一緒にお薬について確認させていただきたいのですが、大丈夫ですか？朝・昼・晩と3回ありますが、昨日はすべて飲めましたか？」

※「飲めていますか」ではなく「飲みましたか」と過去の具体的状況を聞くことで、より正確な情報が得られます。

**STEP 2：服薬障害の要因特定**

患者さんが飲み忘れや中断をしている場合、理由を傾聴の姿勢で確認します。

```
患者さんの発言の例        → 看護師の対応例
───────────────────────────────────
「薬の副作用がつらい」    → 具体的な症状を聞き、医師に報告
「効いているか分からない」 → 症状の改善度を一緒に振り返る
「病気だと思わない」      → 疾病認識を深める情報提供
「飲み忘れてしまう」      → 飲用のタイミングを工夫する支援
```

**STEP 3：服薬改善計画の立案**

患者さんの状況に応じて、以下のいずれかまたは組み合わせで対応します：

**①　服用方法の工夫：**
- 朝食後など日常生活の習慣と結びつける
- 1回分をまとめて薬包に小分けする
- スマートフォンのアラーム設定を患者さんと一緒に設定
- 家族に協力してもらう（ただし患者さんの自主性を尊重）

**②　副作用対策：**
- 医師に相談し、用量調整や薬剤変更を検討
- 副作用の一般的な経過を説明（「1～2週間で落ち着くことが多い」など）
- 対症療法（例：便秘に対する下剤の追加）の提案

**③　疾病認識・心理教育：**
- 患者さんが経験した症状の改善を具体的に説明
- 「薬を飲むことで、〇〇さんが安定して生活できている」と視覚化
- 薬を飲まなかった場合の悪化プロセスを一緒に振り返る

### 2-3. 服薬管理の具体的な支援方法

**方法1：薬局との連携強化**

訪問看護師が薬局の薬剤師と連携することで、患者さんの服薬支援が大きく向上します。

```
【実施内容】
・訪問初回時に患者さんの同意を得て、薬局に連絡
・月1回程度、薬剤師から患者さんへの電話相談
・副作用や飲み忘れに関する情報共有
・一包化（複数の薬を1回分ずつパッキング）の提案
・薬の効果・副作用についての「薬の手帳」への記入支援
```

**方法2：訪問看護記録への工夫**

服薬状況を可視化することで、医師・薬剤師との連携が円滑になります。

```
【記録の具体例】

日時：20XX年X月X日 13:00～13:30
患者対応：本人・妻同席

【服薬状況】
・朝食後：毎日飲用できている
・昼食後：週3～4回の飲み忘れあり（患者さんが「昼間は外出」）
・夜寝前：毎日飲用できている
・副作用：「最近、手がちょっと震える。朝、特につらい」

【アセスメント】
統合失調症患者、疾病認識あり、服薬への動機は良好だが、
生活パターンにより昼食時の飲み忘れが常態化している。
新規副作用（振戦）の報告あり。

【対応内容】
・昼食後の服用を「午後3時の水分補給時」に変更するよう提案
・医師に振戦副作用の報告・対応相談を予定
・妻に、患者さんの自主性を尊重しながら声かけする方法を指導

【次回訪問での評価項目】
☐ 昼間の服用パターンが改善したか
☐ 副作用の経過（振戦の有無・程度）
```

**方法3：セルフモニタリングの導入**

患者さん自身が服薬と症状の関連性に気づくことで、主体性が高まります。

```
【セルフモニタリングシート例】

        飲めた日      飲み忘れた日
気分     ☺ ☺         ☹ ☹
寝付き   〇 〇        △ △
不安感   なし         あり
幻聴     なし         あり

患者さんへの問いかけ：
「このシートを見ると、どんなことに気づきますか？」
→ 患者さん自身が「薬を飲むと調子が良い」と気づく効果
```

## 3. 危機介入の実践的スキル

### 3-1. 危機状態の定義と種類

精神疾患患者における危機状態は、患者さんが自らの対処能力を超えた状況であり、自殺企図・暴力・衝動行為に至る可能性がある状態です。

**危機状態の分類：**

| 危機の種類 | 具体例 | 頻度 | 訪問看護での対応難度 |
|-----------|--------|------|-------------|
| **自殺危機** | 自殺念慮・計画・企図 | 高 | 最高 |
| **精神症状の急激な悪化** | 妄想・幻聴の増加、統制不能 | 中 | 高 |
| **攻撃性の顕在化** | 暴力・暴言・器物破損 | 中 | 高 |
| **社会的危機** | 経済困窮・housing insecurity | 中 | 中 |
| **身体的危機** | 転倒・服用過剰・栄養不良 | 中 | 中 |

### 3-2. 自殺リスク評価の標準手順

訪問看護において自殺リスクを見落とさないことが重要です。以下の手順に基づいて評価します。

**段階1：危険信号の認識**

訪問時に以下の危険信号を察知したら、即座にリスク評価を開始します。

```
【言語的危険信号】
☐ 「死にたい」「消えたい」という言葉
☐ 「自分がいなくなった方がいい」
☐ 「迷惑をかけている」という自責感の表出
☐ 「今夜、決めようと思う」など具体性のある言及

【行動的危険信号】
☐ 自宅に引きこもり、外出拒否の悪化
☐ 日常生活行動の急激な低下（食事摂取、入浴など）
☐ 遺言状・エンディングノート作成
☐ 医師の指示や訪問看護師のアドバイスへの無関心
☐ 所有物の処分

【精神症状の危険信号】
☐ うつ状態の急激な悪化
☐ 妄想内容に「死」が含まれる
☐ 神経過敏性の著しい増加
```

危険信号を察知したら、以下の対応を実施します。

3. 十分な時間確保：「今日は十分な時間がありますので」と伝える
4. 非批判的態度：「そんなこと言うなよ」はNG
5. 傾聴に徹する：助言よりも「聞く」を優先
